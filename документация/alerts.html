Уведомления

Уведомления используются для привлечения внимания пользователя к важной информации. Photon включает статические алерты и динамические тосты с JavaScript API.

Алерты

Алерты — это стилизованные блоки с сообщениями. Используйте класс alert с цветовыми модификаторами: alert--primary, alert--success, alert--warning, alert--error, alert--info.

Основное уведомление с полезной информацией.
Действие выполнено успешно! Все изменения сохранены.
Внимание! Ваша сессия истекает через 5 минут.
Ошибка! Не удалось сохранить изменения. Попробуйте снова.
Новое обновление доступно для загрузки. 
<div class="alert alert--primary">Основное уведомление.</div>
<div class="alert alert--success">Действие выполнено успешно!</div>
<div class="alert alert--warning">Внимание! Сессия истекает.</div>
<div class="alert alert--error">Ошибка! Не удалось сохранить.</div>
<div class="alert alert--info">Новое обновление доступно.</div>

Алерты с кнопкой закрытия

Добавьте класс alert--dismissible и кнопку с классом alert__close.

Файл успешно загружен!
Ваша подписка истекает через 3 дня.
Произошла сетевая ошибка. Проверьте подключение. 
<div class="alert alert--success alert--dismissible">
  <span>Файл успешно загружен!</span>
  <button class="alert__close" type="button">&times;</button>
</div>
<div class="alert alert--warning alert--dismissible">
  <span>Подписка истекает через 3 дня.</span>
  <button class="alert__close" type="button">&times;</button>
</div>

Тосты (JavaScript)

Для динамических уведомлений используйте глобальную функцию window.pkToast(message, type, duration). Типы: success, warning, error, info. Длительность по умолчанию — 3000 мс.

<!-- Вызов тоста из JavaScript -->
<script>
  pkToast('Успешное сохранение!', 'success');
  pkToast('Внимание! Проверьте данные.', 'warning');
  pkToast('Ошибка соединения.', 'error');
  pkToast('Новое сообщение.', 'info');
  pkToast('Обычный тост', 'primary', 5000); // 5 секунд
</script>

Кроме JS-вызова, тосты можно создавать через HTML:

<div class="toast toast--success">
  <span>Успешное сохранение!</span>
  <button class="toast__close" type="button">&times;</button>
</div>
<div class="toast toast--error">
  <span>Ошибка соединения.</span>
  <button class="toast__close" type="button">&times;</button>
</div>
