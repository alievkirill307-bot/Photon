Значки и теги

Photon включает четыре типа маркировочных элементов: значки (badge), теги (tag), чипсы (chip) и метки (label). Каждый из них может быть разных цветов и размеров.

Значки (Badge)

Значки используются для отображения счётчиков, статусов и коротких меток. Используйте класс badge с цветовыми модификаторами.

Основной Успех Внимание Ошибка Инфо Обычный 
<span class="badge badge--primary">Основной</span>
<span class="badge badge--success">Успех</span>
<span class="badge badge--warning">Внимание</span>
<span class="badge badge--error">Ошибка</span>
<span class="badge badge--info">Инфо</span>
<span class="badge">Обычный</span>

Теги (Tag)

Теги похожи на badges, но с немного другим стилем. Используйте класс tag.

HTML CSS JS React Vue TypeScript Node.js 
<span class="tag tag--primary">HTML</span>
<span class="tag tag--success">CSS</span>
<span class="tag tag--warning">JS</span>
<span class="tag tag--error">React</span>
<span class="tag tag--info">Vue</span>

Чипсы (Chip)

Чипсы представляют собой интерактивные элементы с возможностью закрытия. Используйте класс chip.

JavaScript Python В ожидании Баг Feature 
<span class="chip chip--primary">
  JavaScript <button class="chip__close" type="button">&times;</button>
</span>
<span class="chip chip--success">
  Python <button class="chip__close" type="button">&times;</button>
</span>
<span class="chip chip--warning">
  В ожидании <button class="chip__close" type="button">&times;</button>
</span>

Метки (Label)

Метки предназначены для маркировки полей форм и других элементов интерфейса.

Новое Активно Ожидание Критично Инфо Черновик 
<span class="label label--primary">Новое</span>
<span class="label label--success">Активно</span>
<span class="label label--warning">Ожидание</span>
<span class="label label--error">Критично</span>
<span class="label label--info">Инфо</span>
<span class="label">Черновик</span>

Значки в контексте кнопок

Значки удобно размещать внутри кнопок для отображения счётчиков.

<button class="btn btn--primary">
  Уведомления <span class="badge" style="background:rgba(255,255,255,0.2); margin-left:0.4rem;">12</span>
</button>
<button class="btn btn--outline">
  Сообщения <span class="badge badge--primary" style="margin-left:0.4rem;">5</span>
</button>
