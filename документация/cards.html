Карточки

Карточки — универсальный контейнер для группировки контента. Photon поддерживает обычные карточки, стеклянные, с заголовком и подвалом, а также сетки карточек.

Обычная карточка

Базовая карточка с классом card. Содержит тень, скруглённые углы и отступы.

Это простая карточка. Она может содержать любой контент: текст, изображения, кнопки.

Ещё одна карточка для примера расположения в ряд.

<div class="card" style="max-width: 300px;">
  <p style="margin: 0;">Это простая карточка. Она может содержать любой контент.</p>
</div>

Карточка с заголовком и подвалом

Используйте card__header, card__body и card__footer для структурирования содержимого.

Заголовок карточки

Основной контент карточки. Здесь может быть текст, изображения, списки или другие элементы.

<div class="card" style="max-width: 350px;">
  <div class="card__header">
    <h3 style="margin: 0; font-size: 1.1rem;">Заголовок карточки</h3>
  </div>
  <div class="card__body">
    <p style="margin: 0;">Основной контент карточки.</p>
  </div>
  <div class="card__footer" style="display: flex; gap: 0.5rem;">
    <button class="btn btn--primary btn--sm">Подробнее</button>
    <button class="btn btn--outline btn--sm">Отмена</button>
  </div>
</div>

Стеклянная карточка

Добавьте класс card--glass для эффекта матового стекла.

Стеклянная карточка

Эффект матового стекла с размытием фона.

<div class="card card--glass" style="max-width: 300px;">
  <div class="card__header">
    <h3 style="margin: 0; font-size: 1.1rem; color: #fff;">Стеклянная карточка</h3>
  </div>
  <div class="card__body">
    <p style="margin: 0; color: rgba(255,255,255,0.8);">Эффект матового стекла.</p>
  </div>
  <div class="card__footer">
    <button class="btn btn--glass">Действие</button>
  </div>
</div>

Карточка с изображением

Изображение в карточке с классом card__image.

Заголовок

Описание карточки с изображением в верхней части.

  <img class="card__image" src="image.jpg" alt="Изображение" />
  <div class="card__body">
    <h3 style="margin: 0 0 0.5rem;">Заголовок</h3>
    <p style="margin: 0 0 0.75rem; font-size: 0.85rem;">Описание карточки.</p>

  </div>
</div>

Панели и плитки

Photon также включает panel и tile для альтернативных вариантов группировки контента.

Панель

Содержимое панели с заголовком и подвалом.

Подвал панели

Плитка

Плитка с иконкой и контентом.

<div class="panel" style="max-width: 400px;">
  <div class="panel__header">
    <h3 style="margin: 0; font-size: 1rem;">Панель</h3>
  </div>
  <div class="panel__body">
    <p style="margin: 0;">Содержимое панели.</p>
  </div>
  <div class="panel__footer" style="font-size: 0.8rem; color: #6e6e73;">
    Подвал панели
  </div>
</div>

<div class="tile" style="max-width: 400px;">
  <div class="tile__icon">
    <svg viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" style="width:24px;height:24px;color:var(--pk-primary);">
      <circle cx="12" cy="12" r="10"/>
      <path d="M12 16v-4M12 8h.01"/>
    </svg>
  </div>
  <div class="tile__content">

    <p style="margin: 0; font-size: 0.85rem;">Плитка с иконкой и контентом.</p>
  </div>
</div>

Пустое состояние (Empty State)

Компонент для отображения пустого состояния с иконкой, заголовком и действием.

Нет файлов

Загрузите первый файл, чтобы начать работу.

<div class="empty">
  <div class="empty__icon">
    <svg viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="1.5" style="width:48px;height:48px;color:var(--pk-text-muted);">
      <path d="M22 19a2 2 0 01-2 2H4a2 2 0 01-2-2V5a2 2 0 012-2h5l2 3h9a2 2 0 012 2z"/>
    </svg>
  </div>
  <p class="empty__title">Нет файлов</p>
  <p class="empty__text">Загрузите первый файл.</p>
  <button class="btn btn--primary">Загрузить файл</button>
</div>

Hero

Компонент для создания крупных заголовочных секций.

Hero-секция

Крупный заголовок для вашего сайта

  <h1 style="margin: 0 0 0.5rem;">Hero-секция</h1>
  <p style="margin: 0 0 1rem; color: var(--pk-text-secondary);">Крупный заголовок</p>
  <button class="btn btn--primary">Начать</button>
</div>
