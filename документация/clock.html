Часы реального времени

Компонент часов отображает текущее время с автоматическим обновлением каждую секунду. Поддерживает русскую локализацию даты. Работает на чистом JavaScript без внешних зависимостей.

Демо

Часы синхронизированы с системным временем вашего компьютера. Дата отображается на русском языке.

00:00:00
...

HTML-код

Достаточно скопировать эту разметку на страницу и подключить photon-kit.js — часы заработают автоматически. Атрибут data-clock на корневом элементе активирует компонент.

<div class="clock" data-clock>
  <div class="clock__time">00:00:00</div>
  <div class="clock__date">...</div>
</div>

Атрибуты

АтрибутНазначение
data-clockКорневой элемент — активирует часы

Стилизация

Часы используют CSS-переменные Photon. Вы можете легко изменить размер шрифта или цвет через стандартные CSS-свойства:

.clock__time {
  font-size: 3rem;  /* или любой другой размер */
  color: var(--pk-primary);  /* цвет под основную тему */
}
