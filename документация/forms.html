Формы

Формы — основа сбора данных от пользователя. Photon предоставляет стилизованные текстовые поля, выпадающие списки, переключатели, чекбоксы, радио-кнопки, ползунки и состояния валидации.

Текстовые поля

Базовый <input> и <textarea> автоматически стилизуются при наличии классов input и textarea.

<input type="text" class="input" placeholder="Обычное текстовое поле" />
<input type="email" class="input" placeholder="Email" />
<input type="password" class="input" placeholder="Пароль" />
<input type="number" class="input" placeholder="Число" />
<input type="search" class="input" placeholder="Поиск..." />
<textarea class="textarea" placeholder="Многострочный текст"></textarea>

Выпадающие списки

Элемент <select> с классом select.

<select class="select">
  <option>Выберите опцию</option>
  <option>Опция 1</option>
  <option>Опция 2</option>
  <option>Опция 3</option>
</select>
<select class="select" multiple>
  <option>Множественный выбор 1</option>
  <option>Множественный выбор 2</option>
  <option>Множественный выбор 3</option>
</select>

Чекбоксы и радио-кнопки

Используйте стандартные элементы с классами checkbox, radio и обёрткой form-check.

<label class="form-check">
  <input type="checkbox" class="checkbox" checked /> <span>Отмечен</span>
</label>
<label class="form-check">
  <input type="checkbox" class="checkbox" /> <span>Не отмечен</span>
</label>
<label class="form-check">
  <input type="radio" name="r" class="radio" checked /> <span>Радио 1</span>
</label>
<label class="form-check">
  <input type="radio" name="r" class="radio" /> <span>Радио 2</span>
</label>

Переключатели (Switch)

Стилизованные переключатели с классом switch внутри form-check.

<label class="form-check">
  <input type="checkbox" class="switch" checked /> <span>Включено</span>
</label>
<label class="form-check">
  <input type="checkbox" class="switch" /> <span>Выключено</span>
</label>
<label class="form-check">
  <input type="checkbox" class="switch" disabled /> <span>Отключено</span>
</label>

Ползунок (Range)

Стилизованный ползунок выбора диапазона.

<input type="range" class="range" min="0" max="100" value="50" />
<input type="range" class="range range--success" min="0" max="100" value="75" />
<input type="range" class="range range--error" min="0" max="100" value="25" />

Загрузка файлов

Стилизованный input для загрузки файлов.

<input type="file" class="input" />
<input type="file" class="input" multiple />

Валидация

Добавьте класс input--success или input--error для отображения состояния валидации.

<input type="text" class="input input--success" placeholder="Успех" value="Верный Email" />
<input type="text" class="input input--error" placeholder="Ошибка" value="неверный-email" />
<textarea class="textarea input--success">Всё верно</textarea>

Группа формы

Объедините метку, поле и подсказку с помощью классов form-group и form-label.

Минимум 3 символа
Введите корректный email 
<div class="form-group">
  <label class="form-label">Имя пользователя</label>
  <input type="text" class="input" placeholder="Введите имя" />
  <span class="form-hint">Минимум 3 символа</span>
</div>
<div class="form-group">
  <label class="form-label">Email</label>
  <input type="email" class="input input--error" placeholder="Email" value="некорректный" />
  <span class="form-hint form-hint--error">Введите корректный email</span>
</div>

Полная форма

Пример полноценной формы с различными элементами.

<form style="width:100%;max-width:500px;">
  <div class="form-group">
    <label class="form-label">Полное имя</label>
    <input type="text" class="input" placeholder="Иван Иванов" />
  </div>
  <div class="form-group">
    <label class="form-label">Email</label>
    <input type="email" class="input" placeholder="ivan@example.com" />
  </div>
  <div class="form-group">
    <label class="form-label">Страна</label>
    <select class="select">
      <option>Россия</option>
      <option>Казахстан</option>
      <option>Беларусь</option>
    </select>
  </div>
  <div class="form-group">
    <label class="form-label">Подписка</label>
    <label class="form-check"><input type="checkbox" class="checkbox" checked /> <span>Получать новости</span></label>
    <label class="form-check"><input type="checkbox" class="checkbox" /> <span>Спецпредложения</span></label>
  </div>
  <div class="form-group">
    <label class="form-label">Уведомления</label>
    <label class="form-check"><input type="checkbox" class="switch" checked /> <span>Push-уведомления</span></label>
  </div>
  <div class="form-group">
    <label class="form-label">Комментарий</label>
    <textarea class="textarea" placeholder="Ваш комментарий"></textarea>
  </div>
  <button type="submit" class="btn btn--primary">Отправить</button>
  <button type="reset" class="btn btn--outline">Сбросить</button>
</form>
