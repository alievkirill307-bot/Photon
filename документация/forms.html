Формы

Формы — основа сбора данных от пользователя. Photon предоставляет стилизованные текстовые поля, выпадающие списки, переключатели, чекбоксы, радио-кнопки, ползунки, загрузку файлов и состояния валидации.

Текстовые поля

Базовый <input> и <textarea> автоматически стилизуются при наличии классов input и textarea.

<input type="text" class="input" placeholder="Обычное текстовое поле" />
<input type="email" class="input" placeholder="Email" />
<input type="password" class="input" placeholder="Пароль" />
<input type="number" class="input" placeholder="Число" />
<input type="search" class="input" placeholder="Поиск..." />
<textarea class="textarea" placeholder="Многострочный текст"></textarea>

Выпадающие списки

Элемент <select> с классом select.

<select class="select">
  <option>Выберите опцию</option>
  <option>Опция 1</option>
  <option>Опция 2</option>
</select>
<select class="select" multiple>
  <option>Множественный выбор 1</option>
</select>

Чекбоксы и радио-кнопки

Используйте стандартные элементы с классами checkbox, radio и обёрткой form-check.

<label class="form-check">
  <input type="checkbox" class="checkbox" checked /> <span>Отмечен</span>
</label>
<label class="form-check">
  <input type="radio" name="r" class="radio" checked /> <span>Радио 1</span>
</label>

Переключатели (Switch)

Стилизованные переключатели с классом switch внутри form-check.

<label class="form-check">
  <input type="checkbox" class="switch" checked /> <span>Включено</span>
</label>
<label class="form-check">
  <input type="checkbox" class="switch" /> <span>Выключено</span>
</label>

Ползунок (Range)

Стилизованный ползунок выбора диапазона.

<input type="range" class="range" min="0" max="100" value="50" />
<input type="range" class="range" min="0" max="100" value="75" />
<input type="range" class="range" min="0" max="100" value="25" />

Регистрация

Несколько вариантов формы регистрации: простая, расширенная и в стеклянной карточке. Все поля с валидацией через JavaScript.

Вариант 1 — Простая регистрация

Базовая форма: имя, email, пароль, подтверждение пароля и согласие с условиями.

  <div class="form-group">
    <label class="form-label">Имя</label>
    <input type="text" class="input"
      data-validate="required"
      data-validate-length="2" />
    <span class="form-hint form-hint--error"></span>
  </div>
  <div class="form-group">
    <label class="form-label">Email</label>
    <input type="email" class="input"
      data-validate="required email" />
    <span class="form-hint form-hint--error"></span>
  </div>
  <div class="form-group">
    <label class="form-label">Логин</label>
    <input type="text" class="input"
      data-validate="required"
      data-validate-length="3" />
    <span class="form-hint form-hint--error"></span>
  </div>
  <div class="form-row">
    <div class="form-group">
      <label class="form-label">Пароль</label>
      <input type="password" class="input"
        data-validate="required"
        data-validate-length="6" />
    </div>
    <div class="form-group">
      <label class="form-label">Подтверждение</label>
      <input type="password" class="input"
        data-validate="required match"
        data-match-field="Пароль" />
    </div>
  </div>
  <div class="form-group">
    <label class="form-check">
      <input type="checkbox" class="checkbox"
        data-validate="required" />
      <span>Я принимаю условия</span>
    </label>
  </div>
  <button type="submit" class="btn btn--primary
    pk-form-submit">Создать аккаунт</button>
  <div class="pk-form-success alert alert--success"
    style="display:none;">✓ Аккаунт создан!</div>
</form>

JavaScript-валидация форм

Photon имеет встроенную клиентскую валидацию. Добавьте атрибут data-validate к полям и класс pk-form к форме. Доступные правила:

  • required — поле обязательно
  • email — проверка формата email
  • match — совпадение с другим полем (укажите data-match-field="Название")
  • data-validate-length="N" — минимальная длина
<!-- Активация: просто добавьте класс pk-form к форме -->
<form class="pk-form">
  <input type="text" class="input"
    data-validate="required"
    data-validate-length="2" />
  <input type="email" class="input"
    data-validate="required email" />
  <input type="password" class="input"
    data-validate="required match"
    data-match-field="Пароль" />
  <input type="checkbox" class="checkbox"
    data-validate="required" />
  <button type="submit"
    class="btn btn--primary pk-form-submit">
    Отправить
  </button>
  <div class="pk-form-success alert alert--success"
    style="display:none;">✓ Успех!</div>
</form>

Валидация (статическая)

Добавьте класс input--success или input--error для отображения состояния валидации вручную.

<input type="text" class="input input--success"
  placeholder="Успех" value="Верный Email" />
<input type="text" class="input input--error"
  placeholder="Ошибка" value="неверный-email" />
<textarea class="textarea input--success">Всё верно</textarea>
