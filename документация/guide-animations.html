Анимации [guide]

Анимации делают интерфейс живым и отзывчивым. Photon предоставляет набор утилит для CSS-переходов (transitions) и ключевых кадров (keyframes), а также рекомендации по оптимизации производительности.

CSS Transitions

CSS-переходы позволяют плавно изменять значения свойств при смене состояния элемента. Основные параметры: transition-property, transition-duration, transition-timing-function, transition-delay.

Классы Photon для переходов

Используйте pt-transition для базового перехода (all 0.2s ease) и модификаторы для точной настройки.

/* Базовый переход */
.pt-transition {
  transition: all 0.2s ease;
}

/* Покомпонентные классы */
.pt-transition--colors {
  transition: background-color 0.2s ease, color 0.2s ease, border-color 0.2s ease;
}
.pt-transition--transform {
  transition: transform 0.2s ease;
}
.pt-transition--opacity {
  transition: opacity 0.2s ease;
}
.pt-transition--shadow {
  transition: box-shadow 0.2s ease;
}

CSS Animations (keyframes)

Для более сложных анимаций используйте правило @keyframes. Оно определяет последовательность состояний элемента во времени.

@keyframes spin {
  from { transform: rotate(0deg); }
  to   { transform: rotate(360deg); }
}

@keyframes ping {
  0%   { transform: scale(1); opacity: 1; }
  100% { transform: scale(2); opacity: 0; }
}

@keyframes pulse {
  0%, 100% { opacity: 1; }
  50%      { opacity: 0.5; }
}

@keyframes bounce {
  0%, 100% { transform: translateY(0); }
  50%      { transform: translateY(-25%); }
}

Transform в анимациях

Свойство transform идеально подходит для анимаций, так как его изменение не вызывает перекомпоновку (reflow) страницы. Комбинируйте transform с transition для плавных эффектов.

Hover-эффекты

Scale
Rotate
Translate
<div class="pt-transition pt-duration-200"
     onmouseover="this.style.transform='scale(1.15)'"
     onmouseout="this.style.transform='scale(1)'">Scale</div>

<div class="pt-transition pt-duration-200"
     onmouseover="this.style.transform='rotate(10deg)'"
     onmouseout="this.style.transform='rotate(0deg)'">Rotate</div>

<div class="pt-transition pt-duration-200"
     onmouseover="this.style.transform='translateY(-8px)'"
     onmouseout="this.style.transform='translateY(0)'">Translate</div>

Производительность анимаций

Чтобы анимации работали плавно (60 fps), следуйте этим рекомендациям:

will-change

Подскажите браузеру, какие свойства будут анимироваться, с помощью will-change:

.animated-element {
  will-change: transform, opacity;
}

transform vs position

Анимация transform и opacity обрабатывается на GPU и не вызывает reflow. Анимация top, left, width, height вызывает перекомпоновку:

/* Хорошо — GPU-ускорение */
.element {
  transition: transform 0.3s ease;
}
.element:hover {
  transform: translateX(20px);
}

/* Плохо — reflow */
.element {
  transition: left 0.3s ease;
  position: relative;
}
.element:hover {
  left: 20px;
}

GPU-ускорение

Для принудительного GPU-ускорения используйте translateZ(0) или will-change: transform:

.gpu-accelerated {
  transform: translateZ(0);
  /* или */
  will-change: transform;
}
