Blend modes [guide]

CSS blend modes позволяют управлять тем, как цвета элементов смешиваются с фоном или друг с другом. Photon предоставляет утилиты pt-mix-blend-* и pt-bg-blend-* для всех стандартных режимов смешивания, а также pt-isolation-auto и pt-isolate для управления контекстом наложения.

mix-blend-mode

Свойство mix-blend-mode определяет, как содержимое элемента смешивается с содержимым родительского элемента и его фоном. Используйте классы pt-mix-blend-* для быстрого применения.

КлассЗначениеОписание
.pt-mix-blend-normalnormalОбычное наложение, никакого смешивания
.pt-mix-blend-multiplymultiplyУмножение цветов — результат всегда темнее
.pt-mix-blend-screenscreenОбратное умножению — результат всегда светлее
.pt-mix-blend-overlayoverlayКомбинация multiply и screen в зависимости от базового цвета
.pt-mix-blend-darkendarkenВыбирает более тёмный цвет из двух
.pt-mix-blend-lightenlightenВыбирает более светлый цвет из двух
.pt-mix-blend-color-dodgecolor-dodgeОсветляет базовый цвет, уменьшая контраст
.pt-mix-blend-color-burncolor-burnЗатемняет базовый цвет, увеличивая контраст
.pt-mix-blend-hard-lighthard-lightЖёсткий свет — работает как overlay, но roles swapped
.pt-mix-blend-soft-lightsoft-lightМягкий свет — более щадящая версия hard-light
.pt-mix-blend-differencedifferenceРазница — вычитает тёмный цвет из светлого
.pt-mix-blend-exclusionexclusionПохоже на difference, но с меньшим контрастом
.pt-mix-blend-huehueПрименяет оттенок исходного цвета к насыщенности и яркости базового
.pt-mix-blend-saturationsaturationПрименяет насыщенность исходного цвета
.pt-mix-blend-colorcolorПрименяет оттенок и насыщенность исходного цвета к яркости базового
.pt-mix-blend-luminosityluminosityПрименяет яркость исходного цвета к оттенку и насыщенности базового

</div>

Практические примеры

Текст поверх изображения

Используйте pt-mix-blend-overlay или pt-mix-blend-soft-light для создания читаемого текста на изображениях:

<div style="position:relative;">
  <img src="hero.jpg" alt="Hero" style="width:100%;display:block;">
  <div class="pt-mix-blend-overlay"
       style="position:absolute;inset:0;display:flex;align-items:center;justify-content:center;
              background:rgba(0,0,0,0.3);color:#fff;font-size:3rem;font-weight:800;">
    Заголовок
  </div>
</div>

Duotone эффект

Эффект двухцветного изображения (duotone) легко достигается комбинацией градиента и pt-bg-blend-multiply:

.duotone {
  background: linear-gradient(135deg, #ff6b6b, #4ecdc4), url(photo.jpg);
  background-blend-mode: multiply;
  background-size: cover;
}

Цветовые наложения

Цветные оверлеи с разными режимами смешивания позволяют создавать уникальные визуальные стили:

<div class="pt-bg-blend-color" style="background:linear-gradient(135deg,hsl(0,100%,50%),hsl(120,100%,50%)),url(photo.jpg);background-size:cover;">
  Цветной оверлей
</div>

Изоляция

Свойство isolation предотвращает смешивание элемента с его фоном. Photon предоставляет два класса: pt-isolation-auto (стандартное поведение) и pt-isolate (создаёт новый контекст наложения).

/* Изоляция — элемент не смешивается с фоном */
.isolated {
  isolation: isolate;
}

/* Photon-классы */
.pt-isolation-auto { isolation: auto; }
.pt-isolate { isolation: isolate; }

Изоляция полезна, когда нужно ограничить область действия blend mode, чтобы он не распространялся на фоновые элементы.
