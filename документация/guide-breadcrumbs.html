Breadcrumbs (хлебные крошки) [guide]

Breadcrumbs (хлебные крошки) — это элемент навигации, показывающий путь от главной страницы до текущей. Они помогают пользователям понимать своё местоположение и быстро переходить на уровень выше. Photon предоставляет все необходимые утилиты для создания семантически правильных и стильных breadcrumbs.

Структура breadcrumbs

Семантически правильная структура breadcrumbs строится на тегах <nav>, <ol> и <li> со ссылками внутри. Это обеспечивает доступность и правильную индексацию поисковиками.

<nav aria-label="Breadcrumb">
  <ol>
    <li><a href="/">Главная</a></li>
    <li><a href="/category">Категория</a></li>
    <li>Текущая страница</li>
  </ol>
</nav>

Базовая реализация

Используйте утилиты Photon-tw для стилизации: pt-flex, pt-items-center, pt-gap-2, pt-text-sm, pt-text-gray-400 и pt-hover:text-white.

  1. Главная
  2. /
  3. Компоненты
  4. /
  5. Кнопки
<nav aria-label="Breadcrumb">
  <ol class="pt-flex pt-items-center pt-gap-2 pt-list-none pt-p-0 pt-m-0 pt-text-sm pt-text-gray-400">
    <li><a href="#" class="pt-text-gray-400 pt-hover:text-white pt-no-underline pt-transition">Главная</a></li>
    <li class="pt-text-gray-500">/</li>
    <li><a href="#" class="pt-text-gray-400 pt-hover:text-white pt-no-underline pt-transition">Компоненты</a></li>
    <li class="pt-text-gray-500">/</li>
    <li class="pt-text-white pt-font-medium">Кнопки</li>
  </ol>
</nav>

Разделители

В качестве разделителей между элементами можно использовать разные символы: слэш, уголок, стрелку, точку. Выбор разделителя влияет на визуальное восприятие навигации.

РазделительКодПрименение
Слэш //Классический, подходит для большинства сайтов
Уголок >&gt;Современный, часто в админ-панелях
Стрелка →&rarr;Акцент на направлении
Треугольник ▸&blacktriangleright;Визуально выделяется
Точка ·&middot;Минималистичный стиль
Черта |&#124;Вертикальный разделитель
  1. Главная
  2. >
  3. Блог
  4. >
  5. Статья
  1. Главная
  2. →
  3. Каталог
  4. →
  5. Товар
<!-- Разделитель > -->
<li class="pt-text-gray-500">&gt;</li>

<!-- Разделитель → -->
<li class="pt-text-gray-500">&rarr;</li>

Активный элемент

Текущая страница должна быть не ссылкой, а обычным текстом. Выделите её с помощью pt-font-semibold и pt-text-primary (или pt-text-white для тёмной темы).

  1. Главная
  2. /
  3. Профиль
  4. /
  5. Настройки
  6. /
  7. Безопасность
<li class="pt-font-semibold pt-text-primary">Безопасность</li>

Иконки в breadcrumbs

Можно добавить иконку домика перед первым элементом или стрелочки между ссылками. Используйте inline SVG для поддержки тёмной темы.

  1. Главная
  3. Категория
  5. Страница
<!-- Иконка домика -->
<a href="#" class="pt-flex pt-items-center pt-gap-1">
  <svg width="14" height="14" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2">
    <path d="M3 12l9-9 9 9"/>
    <path d="M5 10v10a1 1 0 0 0 1 1h3v-5h6v5h3a1 1 0 0 0 1-1V10"/>
  </svg>
  Главная
</a>

<!-- Иконка-стрелка между элементами -->
<li class="pt-text-gray-500">
  <svg width="12" height="12" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2">
    <path d="M9 18l6-6-6-6"/>
  </svg>
</li>

Адаптивные breadcrumbs

На мобильных устройствах можно скрыть часть элементов, оставив только первый и последний. Используйте pt-hidden и pt-sm:flex для контроля видимости.

  1. Главная
  2. /
  3. Раздел
  4. /
  5. Подраздел
  6. /
  7. Категория
  8. /
  9. Текущая

На мобильных устройствах отображаются только первый и последний элементы

<!-- Элемент скрыт на мобильных, показан на sm+ -->
<li class="pt-hidden pt-sm:inline">
  <a href="#">Подраздел</a>
</li>

<!-- Элемент скрыт на sm, показан на md+ -->
<li class="pt-hidden pt-md:inline">
  <a href="#">Категория</a>
</li>

Schema.org разметка

Для SEO добавьте JSON-LD разметку BreadcrumbList. Это улучшает отображение в поисковых результатах в виде расширенных сниппетов.

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "BreadcrumbList",
  "itemListElement": [
    {
      "@type": "ListItem",
      "position": 1,
      "name": "Главная",
      "item": "https://example.com/"
    },
    {
      "@type": "ListItem",
      "position": 2,
      "name": "Категория",
      "item": "https://example.com/category"
    },
    {
      "@type": "ListItem",
      "position": 3,
      "name": "Текущая страница",
      "item": "https://example.com/category/page"
    }
  ]
}
</script>

Параметры JSON-LD разметки:

ПолеТипОписание
@contextURLВсегда https://schema.org
@typeStringBreadcrumbList
itemListElementArrayМассив элементов, каждый с @type: ListItem
positionNumberПозиция элемента, начиная с 1
nameStringНазвание элемента
itemURLПолный URL страницы (необязательно для последнего)
