CSS Grid [guide]

CSS Grid — мощный инструмент для создания двумерных макетов. Photon предоставляет полный набор утилитарных классов для работы с Grid, включая pt-grid, pt-grid-cols-*, pt-grid-rows-*, pt-gap-* и pt-col-span-*. Эти классы позволяют строить сложные адаптивные сетки без написания CSS.

Основы Grid

Чтобы создать Grid-контейнер, добавьте класс pt-grid к любому элементу. Он устанавливает display: grid. Для управления количеством колонок используйте pt-grid-cols-{n}, где n — число от 1 до 12. Класс pt-grid автоматически включает grid-template-columns: repeat(1, 1fr) как базовое значение.

<div class="pt-grid pt-grid-cols-3">
  <div>Колонка 1</div>
  <div>Колонка 2</div>
  <div>Колонка 3</div>
</div>

Промежутки (pt-gap-*)

Классы для управления расстоянием между Grid-ячейками. Используйте pt-gap-{size} для одинакового отступа по обеим осям, pt-gap-x-{size} для колонок и pt-gap-y-{size} для строк. Значения size соответствуют шкале отступов Photon: 0 (0px), 0.5 (2px), 1 (4px), 2 (8px), 3 (12px), 4 (16px), 5 (20px), 6 (24px), 8 (32px), 10 (40px), 12 (48px), 16 (64px).

КлассCSS значение
.pt-gap-0gap: 0
.pt-gap-0.5gap: 2px
.pt-gap-1gap: 4px
.pt-gap-2gap: 8px
.pt-gap-4gap: 16px
.pt-gap-6gap: 24px
.pt-gap-8gap: 32px
.pt-gap-x-4column-gap: 16px
.pt-gap-y-4row-gap: 16px

Адаптивная сетка

Используйте брейкпоинты Photon для создания адаптивных сеток. Например, pt-grid-cols-1 pt-md:grid-cols-2 pt-lg:grid-cols-3 создаёт одну колонку на мобильных, две на планшетах и три на десктопах.

<div class="pt-grid pt-grid-cols-1 pt-md:grid-cols-2 pt-lg:grid-cols-3 pt-gap-4">
  <div>Элемент 1</div>
  <div>Элемент 2</div>
  <div>Элемент 3</div>
</div>
xs: 1 колонка
md: 2 колонки
lg: 3 колонки
<!-- Адаптивные брейкпоинты -->
.pt-grid-cols-1        { grid-template-columns: repeat(1, 1fr); }
.pt-md:grid-cols-2     { grid-template-columns: repeat(2, 1fr); }
.pt-lg:grid-cols-3     { grid-template-columns: repeat(3, 1fr); }
.pt-xl:grid-cols-4     { grid-template-columns: repeat(4, 1fr); }

<!-- Доступные брейкпоинты: sm (640px), md (768px), lg (1024px), xl (1280px), 2xl (1536px) -->
