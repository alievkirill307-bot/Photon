Flexbox [guide]

Flexbox (Flexible Box Layout) — это одномерный метод раскладки, который позволяет гибко распределять пространство между элементами и выравнивать их. Photon предоставляет полный набор утилитарных классов для работы с Flexbox, охватывающий контейнер и все его дочерние элементы.

Flex-контейнер

Используйте pt-flex для создания flex-контейнера с display: flex или pt-inline-flex для строчного варианта.

КлассCSS значение
.pt-flexdisplay: flex
.pt-inline-flexdisplay: inline-flex
Элемент 1
Элемент 2
Элемент 3
<div class="pt-flex" style="gap:8px;">
  <div>Элемент 1</div>
  <div>Элемент 2</div>
  <div>Элемент 3</div>
</div>

Направление (flex-direction)

Управляет направлением главной оси Flexbox. По умолчанию элементы располагаются в строку (row).

КлассCSS значение
.pt-flex-rowflex-direction: row
.pt-flex-row-reverseflex-direction: row-reverse
.pt-flex-colflex-direction: column
.pt-flex-col-reverseflex-direction: column-reverse
Колонка 1
Колонка 2
Колонка 3
<div class="pt-flex pt-flex-col" style="gap:8px;">
  <div>Колонка 1</div>
  <div>Колонка 2</div>
  <div>Колонка 3</div>
</div>

Перенос (flex-wrap)

Управляет переносом элементов на новую строку при нехватке места.

КлассCSS значение
.pt-flex-wrapflex-wrap: wrap
.pt-flex-nowrapflex-wrap: nowrap
.pt-flex-wrap-reverseflex-wrap: wrap-reverse
1
2
3
4
5
6
<div class="pt-flex pt-flex-wrap" style="gap:8px;max-width:300px;">
  <div>1</div> <div>2</div> <div>3</div>
  <div>4</div> <div>5</div> <div>6</div>
</div>

Выравнивание по главной оси

Классы pt-justify-* управляют распределением элементов вдоль главной оси (горизонталь для row, вертикаль для column).

КлассCSS значение
.pt-justify-startjustify-content: flex-start
.pt-justify-endjustify-content: flex-end
.pt-justify-centerjustify-content: center
.pt-justify-betweenjustify-content: space-between
.pt-justify-aroundjustify-content: space-around
.pt-justify-evenlyjustify-content: space-evenly
Левый
Центр
Правый
<div class="pt-flex pt-justify-between">
  <div>Левый</div>
  <div>Центр</div>
  <div>Правый</div>
</div>

Выравнивание по поперечной оси

Классы pt-items-* управляют выравниванием элементов вдоль поперечной оси (вертикаль для row).

КлассCSS значение
.pt-items-startalign-items: flex-start
.pt-items-endalign-items: flex-end
.pt-items-centeralign-items: center
.pt-items-baselinealign-items: baseline
.pt-items-stretchalign-items: stretch
Высота 40
Высота 60
Высота 30
<div class="pt-flex pt-items-center" style="min-height:100px;">
  <div style="height:40px;">Высота 40</div>
  <div style="height:60px;">Высота 60</div>
  <div style="height:30px;">Высота 30</div>
</div>

Выравнивание контента

Классы pt-content-* управляют распределением строк flex-элементов вдоль поперечной оси (работает с flex-wrap).

КлассCSS значение
.pt-content-startalign-content: flex-start
.pt-content-endalign-content: flex-end
.pt-content-centeralign-content: center
.pt-content-betweenalign-content: space-between
.pt-content-aroundalign-content: space-around
.pt-content-stretchalign-content: stretch

flex-1

Рост и сжатие

Классы pt-flex-grow и pt-flex-shrink управляют способностью элемента увеличиваться или уменьшаться.

КлассCSS значение
.pt-flex-growflex-grow: 1
.pt-flex-grow-0flex-grow: 0
.pt-flex-shrinkflex-shrink: 1
.pt-flex-shrink-0flex-shrink: 0
grow
grow-0
grow
<div class="pt-flex">
  <div class="pt-flex-grow">grow (растянется)</div>
  <div class="pt-flex-grow-0">grow-0 (не растянется)</div>
  <div class="pt-flex-grow">grow (растянется)</div>
</div>

Порядок (order)

Классы pt-order-* меняют порядок отображения элементов. По умолчанию order: 0. Значения от 1 до 12, а также first (-9999) и last (9999).

КлассCSS значение
.pt-order-firstorder: -9999
.pt-order-1order: 1
.pt-order-2order: 2
.pt-order-3order: 3
.pt-order-4order: 4
.pt-order-5order: 5
.pt-order-6order: 6
.pt-order-7order: 7
.pt-order-8order: 8
.pt-order-9order: 9
.pt-order-10order: 10
.pt-order-11order: 11
.pt-order-12order: 12
.pt-order-lastorder: 9999
.pt-order-noneorder: 0
Первый (order 3)
Второй (order 1)
Третий (order 2)

Порядок отображения: Второй, Третий, Первый

<div class="pt-flex">
  <div class="pt-order-3">Первый в DOM (order 3)</div>
  <div class="pt-order-1">Второй в DOM (order 1)</div>
  <div class="pt-order-2">Третий в DOM (order 2)</div>
</div>
<!-- Отобразится: Второй, Третий, Первый -->

КлассCSS значение

.pt-self-endalign-self: flex-end
.pt-self-centeralign-self: center
.pt-self-stretchalign-self: stretch
.pt-self-baselinealign-self: baseline

self-center
self-end

</div>

Примеры

Пример 1: Навбар

Типичная навигационная панель с логотипом слева и ссылками справа:

MyApp
Главная О нас Контакты 
<nav class="pt-flex pt-items-center pt-justify-between" style="padding:12px 20px;background:...;border-radius:12px;">
  <div class="pt-flex pt-items-center" style="gap:10px;">
    <div style="width:32px;height:32px;background:var(--pk-primary);border-radius:8px;"></div>
    <span style="font-weight:700;">MyApp</span>
  </div>
  <div class="pt-flex" style="gap:16px;">
    <a href="#">Главная</a>
    <a href="#">О нас</a>
    <a href="#">Контакты</a>
  </div>
</nav>

Пример 4: Sticky Footer

Футер всегда прижат к низу страницы с помощью pt-flex-col и pt-flex-1:

Шапка
Основной контент растягивается, чтобы заполнить пространство
Футер всегда внизу 
<body class="pt-flex pt-flex-col" style="min-height:100vh;">
  <header>Шапка</header>
  <main class="pt-flex-1">Основной контент</main>
  <footer>Футер</footer>
</body>
