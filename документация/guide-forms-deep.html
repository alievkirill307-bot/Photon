Формы: продвинутый уровень [guide]

Формы — ключевой элемент взаимодействия пользователя с интерфейсом. Photon предоставляет гибкие утилиты для стилизации полей, валидации, кастомных селектов, чекбоксов, переключателей и других элементов форм.

Группировка полей

Используйте fieldset и legend для логической группировки полей. Photon-утилиты помогают управлять отступами.

<fieldset class="pt-border pt-border-gray-200 pt-dark:pt-border-gray-700 pt-rounded-lg pt-p-4 pt-mb-4">
  <legend class="pt-text-sm pt-font-semibold pt-px-2 pt-text-gray-700 pt-dark:pt-text-gray-300">
    Персональные данные
  </legend>

  <div class="pt-mb-3">
    <label class="pt-block pt-text-sm pt-font-medium pt-mb-1 pt-text-gray-700 pt-dark:pt-text-gray-300">Имя</label>
    <input type="text" class="pt-w-full pt-px-3 pt-py-2 pt-border pt-border-gray-300 pt-dark:pt-border-gray-600 pt-rounded-lg pt-bg-white pt-dark:pt-bg-gray-800 pt-text-gray-900 pt-dark:pt-text-gray-100 pt-text-sm pt-focus:pt-ring-2 pt-focus:pt-ring-primary pt-focus:pt-border-primary pt-outline-none pt-transition pt-duration-200" placeholder="Введите имя">
  </div>

  <div class="pt-mb-3">
    <label class="pt-block pt-text-sm pt-font-medium pt-mb-1 pt-text-gray-700 pt-dark:pt-text-gray-300">Email</label>
    <input type="email" class="pt-w-full pt-px-3 pt-py-2 pt-border pt-border-gray-300 pt-dark:pt-border-gray-600 pt-rounded-lg pt-bg-white pt-dark:pt-bg-gray-800 pt-text-gray-900 pt-dark:pt-text-gray-100 pt-text-sm pt-focus:pt-ring-2 pt-focus:pt-ring-primary pt-focus:pt-border-primary pt-outline-none pt-transition pt-duration-200" placeholder="email@example.com">
  </div>
</fieldset>

Валидация

Используйте псевдоклассы :valid, :invalid для стилизации полей в зависимости от их состояния. Photon предоставляет утилиты для этих состояний.

<input type="email" required
       class="pt-w-full pt-px-3 pt-py-2 pt-border pt-border-gray-300 pt-rounded-lg
              pt-valid:pt-border-success pt-valid:pt-ring-success
              pt-invalid:pt-border-error pt-invalid:pt-ring-error
              pt-focus:pt-ring-2 pt-outline-none pt-transition pt-duration-200"
       placeholder="email@example.com">

<!-- Сообщение об ошибке -->
<p class="pt-text-xs pt-text-error pt-mt-1 pt-hidden pt-peer-invalid:pt-block">
  Пожалуйста, введите корректный email
</p>
