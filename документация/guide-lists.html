Списки [guide]

Списки — один из самых распространённых элементов интерфейса. Photon предоставляет утилитарные классы для стилизации маркированных, нумерованных и кастомных списков. В этом гайде — все варианты с примерами.

Маркированные списки

Для маркированных списков используйте ul с классами pt-list-disc, pt-list-circle или pt-list-square. Эти классы меняют list-style-type.

КлассCSSВид маркера
.pt-list-disclist-style-type: discЗаполненный кружок
.pt-list-circlelist-style-type: circleПустой кружок
.pt-list-squarelist-style-type: squareКвадрат

pt-list-disc

  • Первый элемент
  • Второй элемент
  • Третий элемент

pt-list-circle

  • Первый элемент
  • Второй элемент
  • Третий элемент

pt-list-square

  • Первый элемент
  • Второй элемент
  • Третий элемент
<ul class="pt-list-disc pt-pl-5 pt-space-y-1 pt-text-sm">
  <li>Первый элемент</li>
  <li>Второй элемент</li>
  <li>Третий элемент</li>
</ul>

<ul class="pt-list-circle pt-pl-5 pt-space-y-1 pt-text-sm">...</ul>
<ul class="pt-list-square pt-pl-5 pt-space-y-1 pt-text-sm">...</ul>

Нумерованные списки

Для нумерованных списков используйте ol с классами pt-list-decimal, pt-list-lower-roman, pt-list-upper-alpha и другими.

КлассCSSВид нумерации
.pt-list-decimallist-style-type: decimal1, 2, 3, 4...
.pt-list-lower-romanlist-style-type: lower-romani, ii, iii, iv...
.pt-list-upper-romanlist-style-type: upper-romanI, II, III, IV...
.pt-list-lower-alphalist-style-type: lower-alphaa, b, c, d...
.pt-list-upper-alphalist-style-type: upper-alphaA, B, C, D...
.pt-list-lower-greeklist-style-type: lower-greekα, β, γ, δ...

  2. Пункт второй
  3. Пункт третий

Список без маркеров

Используйте pt-list-none чтобы убрать маркеры. Это удобно для горизонтальных списков, навигации и кастомных списков с иконками.

  • Элемент 1 (без маркера)
  • Элемент 2 (без маркера)
  • Элемент 3 (без маркера)
<ul class="pt-list-none pt-space-y-2">
  <li class="pt-p-2 pt-rounded pt-bg-surface-hover">Элемент</li>
  <li class="pt-p-2 pt-rounded pt-bg-surface-hover">Элемент</li>
</ul>

Горизонтальные списки

Комбинируйте pt-list-none с pt-flex pt-gap-* для создания горизонтальных списков. Идеально для навигации, тегов, чипсов.

  • Главная
  • Блог
  • Контакты
  • О нас
  • HTML
  • CSS
  • JavaScript
  • Python
<!-- Горизонтальная навигация -->
<ul class="pt-list-none pt-flex pt-gap-3 pt-text-sm">
  <li><a href="#" class="pt-text-gray-400 pt-hover:text-white pt-no-underline">Главная</a></li>
  <li><a href="#" class="pt-text-gray-400 pt-hover:text-white pt-no-underline">Блог</a></li>
  <li><a href="#" class="pt-text-gray-400 pt-hover:text-white pt-no-underline">Контакты</a></li>
</ul>

<!-- Горизонтальные теги -->
<ul class="pt-list-none pt-flex pt-gap-2 pt-flex-wrap">
  <li><span class="pt-inline-block pt-px-3 pt-py-1 pt-rounded-full pt-text-xs pt-bg-primary pt-text-white">HTML</span></li>
  <li><span class="pt-inline-block pt-px-3 pt-py-1 pt-rounded-full pt-text-xs pt-bg-success pt-text-white">CSS</span></li>
</ul>

Списки с иконками

Замените стандартные маркеры на кастомные иконки. Используйте pt-list-none и добавляйте SVG или символы перед текстом через pt-flex pt-items-center pt-gap-2.

  • Задача выполнена
  • Код написан
  • В процессе
  • Отменено 
<ul class="pt-list-none pt-space-y-3">
  <li class="pt-flex pt-items-center pt-gap-2 pt-text-sm">
    <svg width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="#22c55e" stroke-width="2">
      <path d="M20 6L9 17l-5-5"/>
    </svg>
    <span>Задача выполнена</span>
  </li>
  <li class="pt-flex pt-items-center pt-gap-2 pt-text-sm">
    <svg width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="#3b82f6" stroke-width="2">
      <path d="M22 12h-4l-3 9L9 3l-3 9H2"/>
    </svg>
    <span>В процессе</span>
  </li>
</ul>

Вложенные списки

Списки могут быть вложенными. Photon сохраняет отступы и стили для вложенных элементов. Используйте pt-pl-* для дополнительного отступа.

  • Фрукты
    • Яблоки
    • Груши
    • Цитрусовые
      • Апельсины
      • Лимоны
      • Мандарины
  • Овощи
    • Морковь
    • Картофель
<ul class="pt-list-disc pt-pl-5 pt-space-y-2 pt-text-sm">
  <li>
    Фрукты
    <ul class="pt-list-circle pt-pl-5 pt-space-y-1 pt-mt-1">
      <li>Яблоки</li>
      <li>
        Цитрусовые
        <ul class="pt-list-square pt-pl-5 pt-space-y-1 pt-mt-1">
          <li>Апельсины</li>
          <li>Лимоны</li>
        </ul>
      </li>
    </ul>
  </li>
  <li>Овощи</li>
</ul>

Стилизованные списки

Создавайте кастомные маркеры с помощью CSS-счётчиков или цветных ::before-псевдоэлементов. Используйте pt-list-none и добавьте свои стили.

<style>
.custom-counter {
  counter-reset: item;
  list-style: none;
  padding-left: 0;
}
.custom-counter li {
  counter-increment: item;
  padding-left: 2rem;
  position: relative;
  margin-bottom: 0.5rem;
}
.custom-counter li::before {
  content: counter(item);
  position: absolute;
  left: 0;
  top: 0;
  width: 1.5rem;
  height: 1.5rem;
  background: var(--pk-primary);
  color: #fff;
  border-radius: 50%;
  display: flex;
  align-items: center;
  justify-content: center;
  font-size: 0.75rem;
  font-weight: 600;
}
</style>

<ol class="custom-counter pt-text-sm">
  <li>Зарегистрируйтесь</li>
  <li>Заполните профиль</li>
  <li>Начните работу</li>
</ol>
  1. Зарегистрируйтесь
  2. Заполните профиль
  3. Начните работу
