Медиа-запросы [guide]

Медиа-запросы — основа адаптивного дизайна. Photon-tw использует префиксную систему для применения стилей на разных разрешениях. Это позволяет писать mobile-first код без написания @media вручную.

Брейкпоинты Photon-tw

Photon-tw использует три брейкпоинта, основанных на min-width (mobile-first). Каждый брейкпоинт имеет префикс, который добавляется к любому утилитарному классу.

БрейкпоинтПрефиксmin-widthТипичные устройства
Small (sm)pt-sm:576pxТелефоны в landscape, большие телефоны
Medium (md)pt-md:768pxПланшеты, маленькие ноутбуки
Large (lg)pt-lg:992pxДесктопы, большие экраны
 
/* Пример: CSS-эквивалент классов Photon-tw */
@screen sm (min-width: 576px)  { .pt-sm\:... { ... } }
@screen md (min-width: 768px)  { .pt-md\:... { ... } }
@screen lg (min-width: 992px)  { .pt-lg\:... { ... } }

Mobile-first подход

В mobile-first подходе базовые стили пишутся для мобильных устройств, а затем переопределяются для больших экранов. Пример: колонка на мобильных занимает 100%, на планшетах — 50%, на десктопах — 33%.

/* Base (mobile): 100% ширины */
.element {
  width: 100%;
}

/* sm (>= 576px): 50% */
@media (min-width: 576px) {
  .element { width: 50%; }
}

/* lg (>= 992px): 33.333% */
@media (min-width: 992px) {
  .element { width: 33.333%; }
}

В Photon-tw то же самое одной строкой:

<div class="pt-w-full pt-sm:w-1/2 pt-lg:w-1/3">
  Адаптивный элемент
</div>

Адаптивная типографика

Изменяйте размер текста в зависимости от экрана. Базовый размер задаётся для мобильных, на больших экранах увеличивается.

КлассРазмерПрименение
pt-text-sm pt-md:text-base pt-lg:text-lg14px → 16px → 18pxОсновной текст
pt-text-lg pt-md:text-xl pt-lg:text-2xl18px → 20px → 24pxПодзаголовки
pt-text-2xl pt-md:text-3xl pt-lg:text-4xl24px → 30px → 36pxЗаголовки h2
pt-text-3xl pt-md:text-4xl pt-lg:text-5xl30px → 36px → 48pxЗаголовки h1

Адаптивный заголовок

Этот текст меняет размер в зависимости от ширины экрана. Уменьшите окно браузера и увидите разницу.

<h2 class="pt-text-2xl pt-md:text-3xl pt-lg:text-4xl pt-font-bold">
  Адаптивный заголовок
</h2>
<p class="pt-text-sm pt-md:text-base pt-lg:text-lg pt-text-gray-400">
  Этот текст меняет размер в зависимости от ширины экрана.
</p>

Адаптивные отступы

Отступы также могут быть адаптивными: на мобильных меньше, на десктопах больше. Это улучшает читаемость на больших экранах.

КлассMobilemd+lg+
pt-p-4 pt-md:p-6 pt-lg:p-816px24px32px
pt-m-2 pt-md:m-48px16px16px
pt-gap-2 pt-md:gap-4 pt-lg:gap-68px16px24px
pt-px-4 pt-lg:px-816px16px32px

Этот блок имеет отступы p-4 на мобильных, p-6 на md и p-8 на lg.

<div class="pt-p-4 pt-md:p-6 pt-lg:p-8 pt-bg-surface pt-rounded-xl pt-border pt-border-default">
  <p>Адаптивные отступы</p>
</div>

Адаптивный display

Скрывайте и показывайте элементы на разных экранах. Это удобно для мобильной навигации, сайдбаров, дополнительной информации.

КлассЭффект
pt-hidden pt-md:flexСкрыт на mobile, показан как flex на md+
pt-block pt-lg:inlineБлочный на mobile, строчный на lg+
pt-flex pt-md:hiddenПоказан как flex на mobile, скрыт на md+
pt-hidden pt-sm:blockСкрыт до sm, показан как блок на sm+
pt-inline-block pt-md:hiddenВиден на mobile, скрыт на планшетах+
Только mobile Только md+ Только lg+

Меняйте размер окна — разные элементы появляются/исчезают

<!-- Только на мобильных -->
<span class="pt-px-3 pt-py-1 pt-rounded pt-text-xs pt-bg-primary pt-text-white pt-block pt-md:hidden">
  Только mobile
</span>

<!-- Только на планшетах и выше -->
<span class="pt-px-3 pt-py-1 pt-rounded pt-text-xs pt-bg-success pt-text-white pt-hidden pt-md:inline-block">
  Только md+
</span>

<!-- Только на десктопах -->
<span class="pt-px-3 pt-py-1 pt-rounded pt-text-xs pt-bg-info pt-text-white pt-hidden pt-lg:inline-block">
  Только lg+
</span>

Комбинирование модификаторов

Префиксы можно комбинировать с другими модификаторами, например pt-hover. Это позволяет менять стили при наведении только на определённых разрешениях.

КомбинацияРезультат
pt-md:pt-hover:bg-primaryФон primary при наведении, только на md+
pt-lg:pt-hover:scale-110Увеличение при наведении, только на lg+
pt-sm:pt-focus:ring-2Кольцо при фокусе, только на sm+
pt-md:pt-active:scale-95Уменьшение при нажатии, только на md+
pt-lg:pt-group-hover:opacity-100Показать при наведении на группу, lg+

На md+ при наведении меняет цвет. На mobile — без изменений (touch-устройства)

<button class="pt-px-4 pt-py-2 pt-rounded-lg pt-text-sm pt-bg-surface pt-border pt-border-default pt-text-gray-300 pt-cursor-pointer pt-transition pt-md:pt-hover:bg-primary pt-md:pt-hover:text-white pt-md:pt-hover:border-primary">
  Наведи (только на md+)
</button>
