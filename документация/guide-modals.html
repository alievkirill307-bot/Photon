Модальные окна [guide]

Модальные окна — это диалоговые окна поверх основного контента. Они блокируют взаимодействие с остальной страницей до закрытия. В этом гайде мы соберём модалки из утилитарных классов Photon.

Структура модалки

Базовая модалка состоит из четырёх частей:

  1. Overlay (фон) — полупрозрачный слой, блокирующий взаимодействие с контентом.
  2. Dialog (диалог) — контейнер с контентом.
  3. Content (контент) — заголовок, тело, действия.
  4. Close button (кнопка закрытия) — крестик для закрытия.
<!-- Overlay -->
<div style="position:fixed;inset:0;z-index:9999;background:rgba(0,0,0,0.6);display:flex;align-items:center;justify-content:center;">
  <!-- Dialog -->
  <div style="background:var(--pk-surface);border-radius:12px;padding:24px;max-width:480px;width:100%;margin:16px;">
    <!-- Header -->
    <div style="display:flex;justify-content:space-between;align-items:center;margin-bottom:16px;">
      <h2 style="font-size:1.25rem;font-weight:600;">Заголовок</h2>
      <button style="background:none;border:none;font-size:1.5rem;cursor:pointer;color:var(--pk-muted);">&times;</button>
    </div>
    <!-- Body -->
    <p>Содержимое модального окна.</p>
    <!-- Footer/Actions -->
    <div style="display:flex;justify-content:flex-end;gap:8px;margin-top:24px;">
      <button class="btn">Отмена</button>
      <button class="btn btn--primary">Подтвердить</button>
    </div>
  </div>
</div>

Управление через JS

Для удобной работы с модалками можно написать простую функцию:

function openModal(id) {
  const modal = document.getElementById(id);
  modal.style.display = 'flex';
  // Анимация появления
  requestAnimationFrame(() => {
    modal.style.opacity = '1';
    const dialog = modal.querySelector('[class*="pt-bg-surface"]');
    if (dialog) dialog.style.transform = 'scale(1)';
  });
}

function closeModal(id) {
  const modal = document.getElementById(id);
  modal.style.opacity = '0';
  const dialog = modal.querySelector('[class*="pt-bg-surface"]');
  if (dialog) dialog.style.transform = 'scale(0.95)';
  setTimeout(() => modal.style.display = 'none', 300);
}

// Закрытие по Escape
document.addEventListener('keydown', (e) => {
  if (e.key === 'Escape') {
    document.querySelectorAll('div[style*="display: flex"][style*="position: fixed"]').forEach(m => {
      closeModal(m.id);
    });
  }
});
