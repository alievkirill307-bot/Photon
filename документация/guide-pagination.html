Pagination [guide]

Пагинация (постраничная навигация) используется для разделения большого объёма контента на страницы. Photon предоставляет утилиты для создания гибкой и адаптивной пагинации с поддержкой всех состояний и размеров.

Структура пагинации

Семантически правильная пагинация использует <nav> с aria-label="Pagination", внутри которого располагается <ul> со списком страниц <li>, каждая из которых содержит <a> или <span>.

<nav aria-label="Pagination">
  <ul>
    <li><a href="?page=1">1</a></li>
    <li><a href="?page=2">2</a></li>
    <li><span>3</span></li>
    <li><a href="?page=4">4</a></li>
  </ul>
</nav>

Базовая пагинация

Используйте pt-flex, pt-items-center, pt-gap-1, pt-list-none для контейнера. Каждую ссылку стилизуйте с помощью pt-px-3 pt-py-1.5 pt-rounded-lg pt-text-sm pt-transition.

Состояния

Пагинация имеет три ключевых состояния: активная (текущая страница), disabled (недоступная кнопка) и hover (наведение).

СостояниеКлассыОписание
Обычная ссылкаpt-text-gray-400 pt-hover:bg-surface-hover pt-hover:text-whiteАктивная, но не текущая страница
Активная (текущая)pt-text-white pt-bg-primaryСтраница, на которой находится пользователь
Disabledpt-text-gray-600 pt-cursor-not-allowed pt-opacity-50Кнопка «Назад» на первой странице и т.п.
Hoverpt-hover:bg-surface-hover pt-hover:text-whiteАвтоматически при наведении

<!-- Disabled -->
<li><span class="pt-px-3 pt-py-1.5 pt-rounded-lg pt-text-sm pt-text-gray-600 pt-cursor-not-allowed pt-opacity-50">&laquo;</span></li>

<!-- Active -->
<li><span class="pt-px-3 pt-py-1.5 pt-rounded-lg pt-text-sm pt-text-white pt-bg-primary">2</span></li>

<!-- Default -->
<li><a href="#" class="pt-px-3 pt-py-1.5 pt-rounded-lg pt-text-sm pt-text-gray-400 pt-hover:bg-surface-hover pt-hover:text-white">3</a></li>

Размеры

Пагинация может быть трёх размеров: маленькая, стандартная и большая. Размеры регулируются через pt-text-* и pt-px-* pt-py-*.

РазмерКлассыПрименение
Маленькая (sm)pt-text-xs pt-px-2 pt-py-1Компактные интерфейсы, сайдбары
Стандартная (md)pt-text-sm pt-px-3 pt-py-1.5Основной вариант для большинства страниц
Большая (lg)pt-text-base pt-px-4 pt-py-2Акцент на навигации, главные страницы

Адаптивная пагинация

На мобильных устройствах показывайте меньше страниц. Используйте pt-hidden pt-sm:inline для скрытия части номеров.

  <a href="#" class="...">3</a>
</li>
