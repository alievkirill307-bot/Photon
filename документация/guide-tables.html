Таблицы [guide]

Таблицы используются для отображения структурированных данных. Photon предоставляет утилиты для создания семантически правильных, адаптивных и стилизованных таблиц с поддержкой сортировки и действий.

Структура таблицы

Семантически правильная таблица состоит из <table>, <thead> (заголовок), <tbody> (тело), <tr> (строка), <th> (ячейка заголовка) и <td> (ячейка данных).

<table>
  <thead>
    <tr>
      <th>Имя</th>
      <th>Возраст</th>
      <th>Город</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Анна</td>
      <td>25</td>
      <td>Москва</td>
    </tr>
  </tbody>
</table>

Базовая таблица

Используйте pt-table-auto (или pt-table-fixed), pt-w-full, pt-text-sm и pt-border для базового стиля. Добавьте pt-border-collapse для схлопывания границ.

Оформление колонок

Выравнивайте текст в колонках с помощью pt-text-left, pt-text-center, pt-text-right. Ограничивайте ширину с помощью pt-w-* или pt-max-w-*.

pt-text-lefttext-align: leftВыравнивание по левому краю (текст, имена)

pt-text-righttext-align: rightВыравнивание по правому краю (цены, числа)
pt-w-1/4width: 25%Фиксированная ширина колонки

Адаптивные таблицы

Для адаптивности используйте контейнер с overflow-x-auto. На мобильных можно превратить таблицу в карточки с помощью медиа-запросов. Это особенно удобно для таблиц с большим количеством колонок.

/* CSS для карточного layout на мобильных */
@media (max-width: 640px) {
  .responsive-table table,
  .responsive-table thead,
  .responsive-table tbody,
  .responsive-table th,
  .responsive-table td,
  .responsive-table tr {
    display: block;
  }
  .responsive-table thead {
    display: none; /* Прячем заголовки */
  }
  .responsive-table tr {
    margin-bottom: 1rem;
    padding: 1rem;
    border: 1px solid var(--pk-border);
    border-radius: 8px;
  }
  .responsive-table td {
    padding: 0.25rem 0;
    text-align: right;
    border: none;
  }
  .responsive-table td::before {
    content: attr(data-label);
    float: left;
    font-weight: 600;
    color: var(--pk-muted);
  }
}
