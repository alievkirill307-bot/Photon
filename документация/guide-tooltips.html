Тултипы [guide]

Тултипы — это небольшие всплывающие подсказки, которые появляются при наведении на элемент. В этом гайде мы создадим тултипы с нуля, используя утилитарные классы Photon и чистый CSS.

CSS-тултипы через pt-* классы

Базовый тултип строится на комбинации pt-relative (родитель) и pt-absolute (подсказка). Родитель является точкой отсчёта для позиционирования.

<div class="pt-relative pt-inline-block">
  <!-- Элемент-триггер -->
  Наведи на меня
  <!-- Тултип -->
  <div class="pt-absolute pt-bottom-full pt-left-1/2 pt-translate-x-neg-1/2 pt-mb-2
              pt-bg-gray-800 pt-text-white pt-text-sm pt-px-2 pt-py-1 pt-rounded
              pt-opacity-0 pt-group-hover-opacity-100 pt-transition pt-pointer-events-none
              pt-whitespace-nowrap">
    Подсказка
  </div>
</div>

Тултип через data-атрибуты

Можно создать тултипы полностью на CSS, используя атрибут data-tooltip и функцию attr(). Это удобно для простых подсказок без дополнительной разметки.

/* Базовый стиль для data-тултипа */
[data-tooltip] {
  position: relative;
  cursor: pointer;
}

[data-tooltip]::after {
  content: attr(data-tooltip);
  position: absolute;
  bottom: 100%;
  left: 50%;
  transform: translateX(-50%);
  margin-bottom: 6px;
  padding: 4px 10px;
  background: #1f2937;
  color: #fff;
  font-size: 0.8rem;
  border-radius: 6px;
  white-space: nowrap;
  opacity: 0;
  pointer-events: none;
  transition: opacity 0.2s ease;
  z-index: 10;
}

[data-tooltip]:hover::after {
  opacity: 1;
}

/* Стрелочка */
[data-tooltip]::before {
  content: '';
  position: absolute;
  bottom: 100%;
  left: 50%;
  transform: translateX(-50%);
  margin-bottom: 0;
  border: 5px solid transparent;
  border-top-color: #1f2937;
  opacity: 0;
  pointer-events: none;
  transition: opacity 0.2s ease;
  z-index: 10;
}

[data-tooltip]:hover::before {
  opacity: 1;
}

Пример с data-tooltip

Наведи на меня 
<span data-tooltip="Текст подсказки">Наведи на меня</span>
