Трансформации [guide]

CSS-трансформации позволяют изменять форму, размер и положение элементов без влияния на соседние объекты. Photon предоставляет утилитарные классы на все случаи.

Scale

Масштабирование элемента. Используйте pt-scale-{value} для равномерного масштаба, pt-scale-x-{value} и pt-scale-y-{value} для отдельных осей.

КлассCSS
.pt-scale-50transform: scale(0.5)
.pt-scale-75transform: scale(0.75)
.pt-scale-90transform: scale(0.9)
.pt-scale-100transform: scale(1)
.pt-scale-110transform: scale(1.1)
.pt-scale-125transform: scale(1.25)
.pt-scale-150transform: scale(1.5)
.pt-scale-x-150transform: scaleX(1.5)
.pt-scale-y-150transform: scaleY(1.5)
 
<div class="pt-scale-75 pt-transition"
     onmouseover="this.classList.replace('pt-scale-75','pt-scale-110')"
     onmouseout="this.classList.replace('pt-scale-110','pt-scale-75')">
</div>

.pt-rotate-neg-90transform: rotate(-90deg)

Комбинирование трансформаций

Несколько трансформаций можно комбинировать в одном классе через пробел. Photon поддерживает множественные трансформации через составные утилиты.

.pt-scale-125-rotate-45 {
  transform: scale(1.25) rotate(45deg);
}

.pt-translate-x-4-rotate-90 {
  transform: translateX(16px) rotate(90deg);
}
Scale + Rotate
Translate + Scale 
<div class="pt-transition"
     onmouseover="this.style.transform='scale(1.25) rotate(45deg)'"
     onmouseout="this.style.transform='scale(1) rotate(0deg)'">
  Scale + Rotate
</div>

3D трансформации

Для 3D-эффектов используйте perspective на родителе и rotateX, rotateY, translateZ на потомках.

Perspective

Задаёт глубину сцены. Добавьте style="perspective: 600px" к контейнеру.

.scene {
  perspective: 600px;
}

.card-3d {
  transition: transform 0.4s ease;
}

.card-3d:hover {
  transform: rotateY(45deg);
}
