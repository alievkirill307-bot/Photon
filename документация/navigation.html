Навигация

Photon предоставляет полный набор навигационных компонентов: навбары, вкладки, хлебные крошки, шаги и пагинацию. Навбары доступны в 8 вариантах и полностью адаптивны.

Навбар — стеклянный (по умолчанию)

Классический стеклянный навбар с блюром. Используйте класс .navbar. Адаптивное меню с анимированным бургером автоматически срабатывает на ширине меньше 768px.

P Photon
Главная Компоненты Документация Примеры

  <a href="#" class="navbar__brand">
    <span class="navbar__brand-icon">P</span>
    Photon
  </a>
  <button class="navbar__toggle" type="button" aria-label="Меню">
    <span class="navbar__toggle-icon">
      <span></span><span></span><span></span>
    </span>
  </button>
  <div class="navbar__menu">
    <a href="#" class="navbar__link active">Главная</a>
    <a href="#" class="navbar__link">Компоненты</a>
    <a href="#" class="navbar__link">Документация</a>
  </div>
  <div class="navbar__actions">

  </div>
</nav>

Вкладки (Tabs)

Вкладки для переключения между разделами. Используйте классы .tabs и .tab. Для полной ширины добавьте .tabs--full.

  <a href="#" class="tab active">Вкладка 1</a>
  <a href="#" class="tab">Вкладка 2</a>
  <a href="#" class="tab">Вкладка 3</a>
</div>

  </span>
</div>

Пагинация (Pagination)

Навигация по страницам с помощью класса .pagination.

← Назад 1 2 3 ... 10 Вперед → 
<div class="pagination">
  <a href="#" class="pagination__prev">← Назад</a>
  <a href="#" class="active">1</a>
  <a href="#">2</a>
  <a href="#">3</a>
  <span class="pagination__ellipsis">...</span>
  <a href="#">10</a>
  <a href="#" class="pagination__next">Вперед →</a>
</div>
