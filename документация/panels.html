Боковая панель

Боковая панель (off-canvas / drawer) — выдвижная панель, которая появляется с края экрана по нажатию кнопки. Photon поддерживает два варианта: панель слева со стеклянным эффектом и панель справа с классическим фоном.

Панель слева (Off-canvas)

Выдвигается слева с эффектом стекла. Используйте атрибут data-offcanvas на кнопке и структуру .off-canvas.

<button class="btn btn--primary" data-offcanvas="myPanel">Открыть</button>

<div class="off-canvas" id="myPanel">
  <div class="off-canvas__overlay" data-offcanvas-close></div>
  <div class="off-canvas__sidebar">
    <div style="display:flex;align-items:center;justify-content:space-between;margin-bottom:1.5rem;padding-bottom:0.75rem;border-bottom:1px solid var(--pk-border);">
      <h3 style="margin:0;font-size:1.1rem;">Заголовок</h3>
      <button data-offcanvas-close type="button" style="background:none;border:none;color:var(--pk-text);font-size:1.4rem;cursor:pointer;padding:0;line-height:1;">&times;</button>
    </div>
    <p>Содержимое панели.</p>
  </div>
</div>

Панель справа (Drawer)

Выдвигается справа с поверхностным фоном. Используйте атрибут data-drawer на кнопке и структуру .drawer.

<button class="btn btn--info" data-drawer="myDrawer">Открыть</button>

<div class="drawer" id="myDrawer">
  <div style="display:flex;align-items:center;justify-content:space-between;margin-bottom:1.5rem;padding-bottom:0.75rem;border-bottom:1px solid var(--pk-border);">
    <h3 style="margin:0;font-size:1.1rem;">Заголовок</h3>
    <button data-drawer-close type="button" style="background:none;border:none;color:var(--pk-text);font-size:1.4rem;cursor:pointer;padding:0;line-height:1;">&times;</button>
  </div>
  <p>Содержимое панели.</p>
</div>
<div class="drawer-overlay"></div>

Атрибуты

АтрибутДействие
data-offcanvasОткрыть/закрыть панель слева (off-canvas) по ID
data-offcanvas-closeЗакрыть ближайшую off-canvas панель
data-drawerОткрыть/закрыть панель справа (drawer) по ID
data-drawer-closeЗакрыть ближайшую drawer панель
