Photon-tw

Photon-tw — это встроенный утилитарный слой Photon Framework, который позволяет верстать непосредственно в HTML через атомарные pt-* классы. В отличие от классических компонентов Photon (кнопки, формы, карточки), где стили инкапсулированы в CSS-классы фреймворка, утилитарный подход даёт полный контроль над каждым CSS-свойством.

Что такое Photon-tw?

Photon-tw — это JavaScript-движок, встроенный в photon-kit.js. Он сканирует DOM, находит все элементы с классами, начинающимися на pt-, парсит их и генерирует соответствующие CSS-правила прямо в <head> страницы. Вам не нужно писать CSS-файлы — вся стилизация происходит через классы в HTML.

<!-- Вместо написания CSS -->
<div class="card">
  <h2 class="card-title">Заголовок</h2>
  <p class="card-text">Текст</p>
</div>

<!-- Вы пишете прямо в HTML: -->
<div class="pt-bg-white pt-rounded-lg pt-shadow pt-p-6">
  <h2 class="pt-text-xl pt-font-bold pt-mb-2">Заголовок</h2>
  <p class="pt-text-gray-600">Текст</p>
</div>

Как это работает

  1. Загрузка страницы — photon-kit.js запускается и вызывает initPT()
  2. Сканирование DOM — движок находит все элементы с классами pt-*
  3. Парсинг — каждый класс разбивается на префикс, модификаторы (hover, dark, sm, md, lg), категорию, цвет, оттенок и значение
  4. Генерация CSS — создаются CSS-правила и впрыскиваются в <style id="pt-styles">
  5. Наблюдение — MutationObserver отслеживает появление новых pt-* классов в DOM и генерирует для них CSS на лету
<!-- Достаточно подключить photon-kit.js -->
<script src="photon-kit/photon-kit.js"></script>
<!-- Все pt-* классы работают автоматически -->

Преимущества

  • Скорость разработки — не нужно переключаться между HTML и CSS-файлами
  • Предсказуемость — каждый класс делает ровно одно CSS-свойство
  • Отсутствие конфликтов — все классы изолированы префиксом pt-
  • Тёмная тема из коробки — модификатор pt-dark:* работает без дополнительных настроек
  • Адаптивность — pt-sm:*, pt-md:*, pt-lg:* для брейкпоинтов
  • Состояния — pt-hover:*, pt-focus:*, pt-active:*
  • Комбинирование — модификаторы можно объединять: pt-dark:pt-hover:bg-primary-600
  • Низкий порог входа — достаточно знать CSS-свойства и их значения
  • Никаких сборок — работает в браузере без Webpack, Vite, PostCSS

Сравнение: компонентный vs утилитарный подход

СтилизацияГотовая тема в CSSКаждое свойство отдельным классом
КастомизацияЧерез CSS-переменныеЧерез комбинацию классов
Размер HTMLМеньше классов, больше CSSБольше классов, меньше CSS
ГибкостьОграничена вариантами компонентаПолная — любой CSS без ограничений
Кривая обученияНужно знать API компонентовНужно знать CSS-свойства

Повторное использованиеОдин класс = готовый компонентГруппа классов = уникальный дизайн

Быстрый старт

Минимальный шаблон HTML-страницы с Photon-tw:

<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Моя страница</title>
  <link rel="stylesheet" href="photon-kit/photon-kit.css" />
</head>
<body class="theme-dark">
  <div class="pt-flex pt-items-center pt-justify-center pt-min-h-screen">
    <div class="pt-text-center pt-bg-white pt-dark:bg-gray-800 pt-p-8 pt-rounded-lg pt-shadow-xl">
      <h1 class="pt-text-3xl pt-font-bold pt-text-gray-900 pt-dark:text-white pt-mb-4">
        Привет, Photon-tw!
      </h1>
      <p class="pt-text-gray-600 pt-dark:text-gray-300">
        Эта страница использует утилитарный движок Photon-tw.
      </p>
    </div>
  </div>
  <script src="photon-kit/photon-kit.js"></script>
</body>
</html>

Просто подключите photon-kit.css и photon-kit.js — и все pt-* классы начнут работать автоматически.
