Уведомления и бейджи [pt]

Уведомления (alerts), бейджи (badges), тосты (toasts) и баннеры, построенные исключительно на pt-* утилитах — без единой кастомной строчки CSS. Комбинируйте pt-bg-*, pt-text-*, pt-rounded-*, pt-p-* и другие классы для создания любых вариантов.

Alert по цвету

Цветные плашки-уведомления с фоновым цветом из палитры (pt-bg-{color}-100) и контрастным текстом (pt-text-{color}-800). Базовые классы: pt-p-4 (внутренний отступ), pt-rounded-lg (скругление):

Primary alert — основное информационное уведомление
Success alert — уведомление об успешной операции
Warning alert — предупреждение о потенциальной проблеме
Error alert — сообщение об ошибке или критической ситуации
Info alert — информационное уведомление общего характера
<div class="pt-p-4 pt-rounded-lg pt-bg-primary-100 pt-text-primary-800 pt-dark:bg-primary-900 pt-dark:text-primary-200">Primary alert</div>
<div class="pt-p-4 pt-rounded-lg pt-bg-success-100 pt-text-success-800 pt-dark:bg-success-900 pt-dark:text-success-200">Success alert</div>
<div class="pt-p-4 pt-rounded-lg pt-bg-warning-100 pt-text-warning-800 pt-dark:bg-warning-900 pt-dark:text-warning-200">Warning alert</div>
<div class="pt-p-4 pt-rounded-lg pt-bg-error-100 pt-text-error-800 pt-dark:bg-error-900 pt-dark:text-error-200">Error alert</div>
<div class="pt-p-4 pt-rounded-lg pt-bg-info-100 pt-text-info-800 pt-dark:bg-info-900 pt-dark:text-info-200">Info alert</div>

Alert с иконкой

Уведомление с иконкой статуса слева. Используется pt-flex, pt-items-center и pt-gap-2 для выравнивания иконки и текста:

✓ Данные успешно сохранены. Все изменения применены. 
<div class="pt-flex pt-items-center pt-gap-2 pt-p-4 pt-rounded-lg pt-bg-success-100 pt-text-success-800 pt-dark:bg-success-900 pt-dark:text-success-200">
  <span class="pt-text-lg pt-font-bold">✓</span>
  <span>Данные успешно сохранены.</span>
</div>

Alert с крестиком (закрытие)

Уведомление с иконкой слева и кнопкой закрытия справа. Кнопка использует pt-opacity-60 и pt-hover:opacity-100 для эффекта появления при наведении:

ℹ Добро пожаловать в Photon Framework! Начните с изучения документации.
! Ваша сессия истекает через 5 минут. Сохраните изменения.
✕ Ошибка подключения к серверу. Проверьте сетевое соединение. 
<div class="pt-flex pt-items-center pt-justify-between pt-p-4 pt-rounded-lg pt-bg-primary-100 pt-text-primary-800">
  <div class="pt-flex pt-items-center pt-gap-2">
    <span>ℹ</span>
    <span>Текст уведомления</span>
  </div>
  <button class="pt-opacity-60 pt-hover:opacity-100 pt-cursor-pointer pt-bg-transparent pt-border-none pt-text-xl">&times;</button>
</div>

Бейджи

Маленькие цветные метки для статусов, категорий и счётчиков. Базовая структура: pt-bg-{color} + pt-text-white + pt-text-xs + pt-px-2 pt-py-0.5 + pt-rounded-full:

Primary Success Warning Error Info Gray Blue Red Green Purple 
<span class="pt-bg-primary pt-text-white pt-text-xs pt-px-2 pt-py-0.5 pt-rounded-full">Primary</span>
<span class="pt-bg-success pt-text-white pt-text-xs pt-px-2 pt-py-0.5 pt-rounded-full">Success</span>
<span class="pt-bg-warning pt-text-dark pt-text-xs pt-px-2 pt-py-0.5 pt-rounded-full">Warning</span>
<span class="pt-bg-error pt-text-white pt-text-xs pt-px-2 pt-py-0.5 pt-rounded-full">Error</span>

<span class="pt-bg-gray-200 pt-text-gray-700 pt-text-xs pt-px-2 pt-py-0.5 pt-rounded-full">Gray</span>

pt-text-xs (маленький) pt-text-sm (средний) pt-text-base (крупный)

Бейджи на кнопках

Счётчик уведомлений на кнопке с помощью абсолютного позиционирования. Кнопка получает pt-relative, а бейдж — pt-absolute с отрицательными отступами -pt-top-2 -pt-right-2:

<button class="pt-relative pt-bg-primary pt-text-white pt-px-4 pt-py-2 pt-rounded">
  Уведомления
  <span class="pt-absolute -pt-top-2 -pt-right-2 pt-bg-error pt-text-white pt-text-xs pt-w-5 pt-h-5 pt-flex pt-items-center pt-justify-center pt-rounded-full">3</span>
</button>

Toast-уведомление

Всплывающий тост с иконкой, заголовком и описанием. Использует pt-shadow-lg для тени, pt-border-l-4 для цветной полосы слева и pt-rounded-lg для скругления:

✓
Готово!
Операция выполнена успешно. Все изменения сохранены.
✕
Ошибка
Не удалось подключиться к серверу. Попробуйте позже.
!
Внимание
Ваша подписка истекает через 3 дня. Продлите сейчас.
<div class="pt-flex pt-items-center pt-gap-3 pt-p-4 pt-rounded-lg pt-shadow-lg pt-bg-white pt-dark:bg-gray-800 pt-max-w-sm pt-border-l-4 pt-border-success">
  <div class="pt-w-8 pt-h-8 pt-bg-success pt-rounded-full pt-flex pt-items-center pt-justify-center pt-text-white pt-font-bold pt-flex-shrink-0">✓</div>
  <div class="pt-flex-1">
    <div class="pt-font-semibold pt-text-sm">Готово!</div>
    <div class="pt-text-gray-500 pt-text-xs">Операция выполнена.</div>
  </div>
  <button class="pt-text-gray-400 pt-hover:text-gray-600 pt-transition pt-cursor-pointer">&times;</button>
</div>

Баннер (растянутый alert)

Полноширинный баннер на всю ширину родительского контейнера. Используется для системных уведомлений, cookie-согласий и важных объявлений. Ключевые классы: pt-w-full, pt-text-center, pt-py-3:

Добро пожаловать! — Photon Framework v2.0 теперь доступен. Узнать больше
Cookie: мы используем файлы cookie для улучшения работы. Принять
Системное уведомление: сервер будет перезагружен в 03:00 по МСК. 
<div class="pt-w-full pt-text-center pt-py-3 pt-bg-primary pt-text-white">
  <span class="pt-font-semibold">Заголовок</span> — текст баннера
</div>
<div class="pt-w-full pt-text-center pt-py-3 pt-bg-warning pt-text-dark">
  <span class="pt-font-semibold">Предупреждение</span> — текст
</div>
<div class="pt-w-full pt-text-center pt-py-3 pt-bg-error pt-text-white">
  <span class="pt-font-semibold">Ошибка</span> — текст
</div>

Тёмная тема

Все компоненты уведомлений и бейджей поддерживают тёмную тему через префикс pt-dark:. Это позволяет адаптировать цвета фона, текста и границ под тёмную тему без дублирования кода:

Alert в тёмной теме

Primary alert — тёмная тема: pt-dark:bg-primary-900 pt-dark:text-primary-200
Success alert — фон становится темнее, текст светлее
Error alert — автоматическая адаптация контраста

Бейджи в тёмной теме

Primary Success Error

</span>
