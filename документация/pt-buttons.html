Кнопки [pt]

Все кнопки ниже собраны исключительно из pt-* утилитарных классов Photon-tw — без единой кастомной строчки CSS. В отличие от компонентного подхода .btn, где внешний вид задаётся одним классом, утилитарный подход даёт полный контроль над каждым CSS-свойством: фон, отступы, скругление, границы, тени, состояния и анимации собираются как конструктор из отдельных классов. Это позволяет создавать уникальные стили кнопок, не выходя из разметки HTML.

Базовые кнопки

Цветные кнопки строятся на фоновых классах pt-bg-{color} и текстовых pt-text-{color}. Стандартный padding pt-px-4 pt-py-2 и скругление pt-rounded дают аккуратную кнопку.

<button class="pt-bg-primary pt-text-white pt-px-4 pt-py-2 pt-rounded">Primary</button>
<button class="pt-bg-success pt-text-white pt-px-4 pt-py-2 pt-rounded">Success</button>
<button class="pt-bg-warning pt-text-dark pt-px-4 pt-py-2 pt-rounded">Warning</button>
<button class="pt-bg-error pt-text-white pt-px-4 pt-py-2 pt-rounded">Error</button>
<button class="pt-bg-info pt-text-white pt-px-4 pt-py-2 pt-rounded">Info</button>

Таблица классов, использованных в базовых кнопках:

Класс CSS-свойство Значение Описание
pt-bg-primary background-color var(--pk-primary) / #5755d9 Фон основного цвета бренда
pt-bg-success background-color var(--pk-success) / #32b643 Фон цвета успеха
pt-bg-warning background-color var(--pk-warning) / #ffb700 Фон цвета предупреждения
pt-bg-error background-color var(--pk-error) / #e85600 Фон цвета ошибки
pt-bg-info background-color var(--pk-info) / #4b8bf5 Фон информационного цвета
pt-text-white color #ffffff Белый цвет текста для контраста
pt-text-dark color #1d1d1f Тёмный цвет текста (для светлых фонов)
pt-px-4 padding-left / padding-right 1rem (16px) Горизонтальный отступ внутри кнопки
pt-py-2 padding-top / padding-bottom 0.5rem (8px) Вертикальный отступ внутри кнопки
pt-rounded border-radius 0.375rem (6px) Стандартное скругление углов

Размеры кнопок

Управляйте размером кнопки через классы pt-px-* и pt-py-*. Чем больше цифра, тем просторнее кнопка. Ниже приведены пять типоразмеров от «extra small» до «extra large»:

<button class="pt-bg-primary pt-text-white pt-px-4 pt-py-2 pt-rounded">base (4&times;2)</button>
<button class="pt-bg-warning pt-text-dark pt-px-6 pt-py-3 pt-rounded">lg (6&times;3)</button>
<button class="pt-bg-error pt-text-white pt-px-8 pt-py-4 pt-rounded">xl (8&times;4)</button>

Скругление (border-radius)

Photon-tw предоставляет семь уровней скругления через pt-rounded-*. Комбинируйте их с цветными кнопками для создания нужного визуального стиля — от строгих прямоугольников до полностью круглых пилюль.

<button class="pt-bg-primary pt-text-white pt-px-4 pt-py-2 pt-rounded-none">rounded-none</button>
<button class="pt-bg-success pt-text-white pt-px-4 pt-py-2 pt-rounded-sm">rounded-sm</button>
<button class="pt-bg-primary pt-text-white pt-px-4 pt-py-2 pt-rounded">rounded</button>
<button class="pt-bg-warning pt-text-dark pt-px-4 pt-py-2 pt-rounded-lg">rounded-lg</button>
<button class="pt-bg-error pt-text-white pt-px-4 pt-py-2 pt-rounded-xl">rounded-xl</button>
<button class="pt-bg-info pt-text-white pt-px-4 pt-py-2 pt-rounded-2xl">rounded-2xl</button>
<button class="pt-bg-primary pt-text-white pt-px-4 pt-py-2 pt-rounded-full">rounded-full</button>
Класс Значение Визуальный результат
pt-rounded-none border-radius: 0 Прямоугольник без скругления
pt-rounded-sm border-radius: 0.125rem (2px) Едва заметное скругление

Outline кнопки

В отличие от filled-кнопок с заливкой фона, outline-кнопки используют только рамку и цвет текста. Это даёт более воздушный, минималистичный вид. Основной паттерн: pt-border pt-border-{color} pt-text-{color}. Для нейтральных вариантов используйте pt-border-gray-400 и pt-text-gray-600.

<button class="pt-border pt-border-primary pt-text-primary pt-px-4 pt-py-2 pt-rounded pt-hover:bg-primary pt-hover:text-white pt-transition">Primary</button>
<button class="pt-border pt-border-success pt-text-success pt-px-4 pt-py-2 pt-rounded pt-hover:bg-success pt-hover:text-white pt-transition">Success</button>
<button class="pt-border pt-border-warning pt-text-warning pt-px-4 pt-py-2 pt-rounded pt-hover:bg-warning pt-hover:text-dark pt-transition">Warning</button>
<button class="pt-border pt-border-error pt-text-error pt-px-4 pt-py-2 pt-rounded pt-hover:bg-error pt-hover:text-white pt-transition">Error</button>
<button class="pt-border pt-border-info pt-text-info pt-px-4 pt-py-2 pt-rounded pt-hover:bg-info pt-hover:text-white pt-transition">Info</button>

Ключевые отличия outline от filled:

Характеристика Filled (заливка) Outline (рамка)
Фон pt-bg-{color} — сплошной цвет Прозрачный (нет pt-bg-*)
Граница Отсутствует (или скрыта) pt-border pt-border-{color}
Цвет текста pt-text-white или контрастный pt-text-{color} — совпадает с рамкой
Hover-эффект (опционально) Темнее фон (pt-hover:bg-{color}-600) Заливка при наведении (pt-hover:bg-{color})

Состояния (hover, focus, active)

Photon-tw поддерживает модификаторы состояний: pt-hover:*, pt-focus:* и pt-active:*. Они позволяют менять любые CSS-свойства при наведении, фокусе и нажатии. Ниже — примеры различных эффектов:

<!-- Изменение цвета фона при наведении -->
<button class="pt-bg-primary pt-text-white pt-px-4 pt-py-2 pt-rounded pt-hover:bg-primary-600 pt-transition pt-transition-colors pt-duration-200">hover:bg-primary-600</button>

<!-- Появление тени при наведении -->
<button class="pt-bg-primary pt-text-white pt-px-4 pt-py-2 pt-rounded pt-hover:shadow-lg pt-transition pt-duration-200">hover:shadow-lg</button>

<!-- Масштабирование при наведении -->
<button class="pt-bg-primary pt-text-white pt-px-4 pt-py-2 pt-rounded pt-hover:scale-105 pt-transition pt-duration-200">hover:scale-105</button>

<!-- Смещение вверх при наведении -->
<button class="pt-bg-primary pt-text-white pt-px-4 pt-py-2 pt-rounded pt-hover:-pt-translate-y-1 pt-transition pt-duration-200">hover:-translate-y-1</button>

<!-- Прозрачность при наведении -->
<button class="pt-bg-primary pt-text-white pt-px-4 pt-py-2 pt-rounded pt-hover:opacity-80 pt-transition pt-duration-200">hover:opacity-80</button>

<!-- Outline → filled при наведении -->
<button class="pt-border pt-border-primary pt-text-primary pt-px-4 pt-py-2 pt-rounded pt-hover:bg-primary pt-hover:text-white pt-transition pt-duration-200">Outline → filled</button>

<!-- Отключённое состояние -->
<button class="pt-bg-gray-300 pt-text-gray-600 pt-px-4 pt-py-2 pt-rounded pt-opacity-50 pt-cursor-not-allowed" disabled>disabled</button>

Таблица модификаторов состояний:

Модификатор Псевдокласс Пример использования Описание
pt-hover:* :hover pt-hover:bg-primary-600 Стили при наведении курсора
pt-focus:* :focus pt-focus:ring-2 pt-focus:ring-primary Стили при фокусе (клавиатура/клик)
pt-active:* :active pt-active:scale-95 Стили в момент нажатия
disabled [disabled] pt-opacity-50 pt-cursor-not-allowed Атрибут + утилиты для неактивного состояния

Анимации и переходы

Добавьте pt-transition или pt-transition-colors / pt-transition-all для плавных анимаций при смене состояний. Длительность и функция плавности настраиваются через pt-duration-* и pt-ease-*. Сравните кнопки без и с transition:

Без transition:
С transition: 
<!-- Без transition — смена стилей мгновенная -->
<button class="pt-bg-primary pt-text-white pt-px-4 pt-py-2 pt-rounded pt-hover:bg-primary-600">Наведи</button>

<!-- С transition — плавное изменение всех анимируемых свойств -->
<button class="pt-bg-primary pt-text-white pt-px-4 pt-py-2 pt-rounded pt-hover:bg-primary-600 pt-transition-all pt-duration-300 pt-ease-in-out">Наведи</button>
<button class="pt-bg-primary pt-text-white pt-px-4 pt-py-2 pt-rounded pt-hover:shadow-lg pt-transition-all pt-duration-300 pt-ease-in-out">Наведи</button>
<button class="pt-bg-primary pt-text-white pt-px-4 pt-py-2 pt-rounded pt-hover:scale-105 pt-transition-all pt-duration-300 pt-ease-in-out">Наведи</button>

Таблица классов для анимации:

Класс CSS-свойство Описание

pt-transition-colors transition: color 0.15s ease, background-color 0.15s ease, border-color 0.15s ease Переход только для цветовых свойств
pt-transition-all transition: all 0.15s ease Переход для всех анимируемых свойств

pt-ease-linear transition-timing-function: linear Равномерная скорость

pt-ease-out transition-timing-function: cubic-bezier(0, 0, 0.2, 1) Замедление в конце

Группы кнопок

Объединяйте кнопки в горизонтальные группы с помощью pt-flex pt-rounded pt-overflow-hidden. Добавляйте разделители через pt-border-l с цветом, совпадающим на тон темнее фона. Вертикальные группы создаются с pt-flex-col и pt-border-t.

Горизонтальная группа через pt-flex pt-rounded pt-overflow-hidden

<div class="pt-flex pt-rounded pt-overflow-hidden">
  <button class="pt-bg-primary pt-text-white pt-px-4 pt-py-2 pt-hover:bg-primary-600 pt-transition">Левый</button>
  <button class="pt-bg-primary pt-text-white pt-px-4 pt-py-2 pt-hover:bg-primary-600 pt-border-l pt-border-primary-600 pt-transition">Центр</button>
  <button class="pt-bg-primary pt-text-white pt-px-4 pt-py-2 pt-hover:bg-primary-600 pt-border-l pt-border-primary-600 pt-transition">Правый</button>
</div>

Группа с разноцветными кнопками (outline + заполненные):

<div class="pt-flex pt-rounded pt-overflow-hidden">
  <button class="pt-bg-success pt-text-white pt-px-4 pt-py-2 pt-hover:bg-success-600 pt-transition">Сохранить</button>
  <button class="pt-bg-warning pt-text-dark pt-px-4 pt-py-2 pt-hover:bg-warning-600 pt-border-l pt-border-warning-600 pt-transition">Редакт.</button>
  <button class="pt-bg-error pt-text-white pt-px-4 pt-py-2 pt-hover:bg-error-600 pt-border-l pt-border-error-600 pt-transition">Удалить</button>
</div>

Кнопка на всю ширину

Класс pt-w-full растягивает кнопку на всю ширину родительского контейнера. Полезно для мобильных интерфейсов, форм и hero-секций. В демонстрации ниже кнопка ограничена шириной pt-max-w-sm для наглядности.

<button class="pt-bg-primary pt-text-white pt-px-4 pt-py-3 pt-rounded pt-w-full pt-max-w-sm pt-hover:bg-primary-600 pt-transition">
  pt-w-full (полная ширина)
</button>

Блок кнопок на всю ширину в группе:

<button class="pt-bg-primary pt-text-white pt-px-4 pt-py-2 pt-rounded pt-w-full pt-hover:bg-primary-600 pt-transition pt-mb-2">Авторизоваться</button>
<button class="pt-bg-success pt-text-white pt-px-4 pt-py-2 pt-rounded pt-w-full pt-hover:bg-success-600 pt-transition pt-mb-2">Зарегистрироваться</button>
<button class="pt-border pt-border-gray-400 pt-text-gray-600 pt-px-4 pt-py-2 pt-rounded pt-w-full pt-hover:bg-gray-100 pt-transition pt-dark:pt-hover:bg-gray-700">Продолжить как гость</button>

Тёмная тема

Модификатор pt-dark:* позволяет задавать стили для тёмной темы. Используйте его для адаптации кнопок: pt-dark:bg-gray-800 pt-dark:text-white pt-dark:border-gray-600. Если тема документа переключена на theme-dark, классы с префиксом pt-dark:* активируются автоматически.

Переключите тему страницы на тёмную/светлую, чтобы увидеть разницу

<button class="pt-bg-gray-200 pt-text-dark pt-px-4 pt-py-2 pt-rounded pt-dark:bg-gray-800 pt-dark:text-white pt-transition">Светлая / Тёмная</button>
<button class="pt-border pt-border-gray-400 pt-text-gray-700 pt-px-4 pt-py-2 pt-rounded pt-dark:border-gray-500 pt-dark:text-gray-200 pt-transition">Outline адаптивная</button>
<button class="pt-bg-primary pt-text-white pt-px-4 pt-py-2 pt-rounded pt-dark:bg-primary-400 pt-dark:text-dark pt-transition">Primary адаптивная</button>

Таблица модификаторов тёмной темы для кнопок:

Модификатор Назначение Пример комбинации
pt-dark:bg-{color} Фон кнопки в тёмной теме pt-dark:bg-gray-800 pt-dark:bg-primary-400
pt-dark:text-{color} Цвет текста в тёмной теме pt-dark:text-white pt-dark:text-gray-200
pt-dark:border-{color} Цвет рамки в тёмной теме pt-dark:border-gray-500 pt-dark:border-primary-300

Комбинации с другими pt-* классами
