Формы [pt]

Photon-tw предоставляет полный набор pt-* классов для стилизации форм: от простых текстовых полей до сложных композиций с иконками и валидацией. Все стили применяются через утилитарные классы — никакого кастомного CSS.

Текстовое поле

Базовое текстовое поле собирается из комбинации pt-border, pt-rounded-md, отступов pt-px-3 pt-py-2 и ширины pt-w-full. Для состояния фокуса используем pt-focus:border-primary.

<input class="pt-border pt-border-gray-300 pt-rounded-md pt-px-3 pt-py-2 pt-w-full pt-text-sm pt-focus:border-primary pt-focus:ring-2 pt-focus:ring-primary-200 pt-transition" placeholder="Введите текст..." style="outline:none;">

Компоненты текстового поля

Класс Назначение
pt-borderДобавляет рамку шириной 1px
pt-border-gray-300Цвет рамки — серый 300
pt-rounded-mdСкругление 6px (medium)
pt-px-3Горизонтальный padding 12px
pt-py-2Вертикальный padding 8px
pt-w-fullШирина 100% контейнера
pt-text-smРазмер текста 0.875rem
pt-focus:border-primaryПри фокусе рамка становится primary
pt-focus:ring-2При фокусе добавляется внешнее свечение
pt-focus:ring-primary-200Цвет внешнего свечения

Выпадающий список (Select)

Select стилизуется теми же классами, что и текстовое поле. Добавляем pt-appearance-none чтобы убрать нативную стрелку, и pt-cursor-pointer для курсора-указателя.

<select class="pt-border pt-border-gray-300 pt-rounded-md pt-px-3 pt-py-2 pt-w-full pt-text-sm pt-bg-white pt-appearance-none pt-cursor-pointer" style="outline:none;">
  <option>Пункт 1</option>
  <option>Пункт 2</option>
  <option>Пункт 3</option>
</select>

Дополнительные модификаторы Select

  • pt-appearance-none — убирает стандартное оформление браузера
  • pt-cursor-pointer — меняет курсор на указатель
  • pt-bg-transparent — прозрачный фон
  • pt-dark:bg-gray-800 — тёмный фон для ночной темы

Чекбокс и Радио-кнопка

Для чекбоксов и радио используем pt-w-4 pt-h-4 для размера, pt-text-primary для цвета галочки и pt-rounded для скругления (только чекбокс). Оборачиваем в pt-flex pt-items-center pt-gap-2 для правильного выравнивания с текстом.

<label class="pt-flex pt-items-center pt-gap-2 pt-cursor-pointer">
  <input type="checkbox" class="pt-w-4 pt-h-4 pt-text-primary pt-border-gray-300 pt-rounded pt-cursor-pointer"> Чекбокс
</label>
<label class="pt-flex pt-items-center pt-gap-2 pt-cursor-pointer">
  <input type="radio" name="radio" class="pt-w-4 pt-h-4 pt-text-primary pt-border-gray-300 pt-cursor-pointer"> Радио
</label>

Распространённые размеры чекбоксов

Поле с иконкой

Иконка внутри поля реализуется через абсолютное позиционирование. Контейнер получает pt-relative, иконка — pt-absolute с отступом слева pt-left-3 и вертикальным центрированием pt-top-1/2 -pt-translate-y-1/2. Само поле получает pt-pl-10 чтобы текст не наезжал на иконку.

🔍 
<div class="pt-relative pt-w-full pt-max-w-sm">
  <span class="pt-absolute pt-left-3 pt-top-1/2 -pt-translate-y-1/2 pt-text-gray-400 pt-text-sm pt-pointer-events-none">🔍</span>
  <input class="pt-border pt-border-gray-300 pt-rounded-md pt-pl-10 pt-pr-3 pt-py-2 pt-w-full pt-text-sm" placeholder="Поиск..." style="outline:none;">
</div>

Вертикальная форма (Login)

Полноценная форма логина собранная исключительно из pt-* классов. Каждое поле — это <label> и <input> с отступами pt-mb-*. Кнопка на всю ширину использует pt-w-full.

<div class="pt-max-w-sm">
  <label class="pt-text-sm pt-font-medium pt-mb-1 pt-block">Email</label>
  <input class="pt-border pt-border-gray-300 pt-rounded-md pt-px-3 pt-py-2 pt-w-full pt-text-sm pt-mb-3" placeholder="mail@example.com" style="outline:none;">
  <label class="pt-text-sm pt-font-medium pt-mb-1 pt-block">Пароль</label>
  <input type="password" class="pt-border pt-border-gray-300 pt-rounded-md pt-px-3 pt-py-2 pt-w-full pt-text-sm pt-mb-4" placeholder="••••••••" style="outline:none;">
  <button class="pt-bg-primary pt-text-white pt-px-4 pt-py-2 pt-rounded pt-w-full pt-hover:bg-primary-600 pt-transition">Войти</button>
</div>

Структура вертикальной формы

Элемент Ключевые классы
Контейнерpt-max-w-sm
Метка (label)pt-text-sm pt-font-medium pt-mb-1 pt-block
Поле вводаpt-border pt-rounded-md pt-px-3 pt-py-2 pt-w-full pt-text-sm pt-mb-3
Кнопкаpt-bg-primary pt-text-white pt-px-4 pt-py-2 pt-rounded pt-w-full

Адаптация под тёмную тему

Для корректного отображения форм в тёмной теме используйте модификатор pt-dark:*. Он автоматически меняет цвета фона, текста, рамок и свечения при активации класса theme-dark на <body>.

Свойство Светлая тема Тёмная тема
Фон поляpt-bg-whitept-dark:bg-gray-800
Текст поляpt-text-gray-900pt-dark:text-gray-100
Рамкаpt-border-gray-300pt-dark:border-gray-600
Меткаpt-text-gray-700pt-dark:text-gray-300

Состояния валидации

Хотя Photon-tw не предоставляет классов валидации, вы можете комбинировать pt-* классы с кастомными состояниями. Например, для ошибки добавляем pt-border-error pt-text-error, для успеха — pt-border-success.

Состояние Дополнительные классы
Ошибкаpt-border-error pt-text-error
Успехpt-border-success pt-text-success
Отключеноpt-opacity-50 pt-cursor-not-allowed
Обязательноеafter:pt-content-['*'] after:pt-text-error
