Изображения [pt]

Утилиты Photon-tw для работы с изображениями: скругление углов, управление масштабированием (object-fit), позиционирование (object-position), тени, аватарки, адаптивная ширина/высота и градиентные оверлеи. Все примеры собраны из pt-* классов без кастомного CSS.

Скругление углов

Используйте pt-rounded-* для разных степеней скругления. Размер каждого блока — 120×120 px. От pt-rounded-sm (2px) до pt-rounded-full (полный круг).

Object-fit

Управление тем, как содержимое заполняет контейнер. Каждый пример — контейнер pt-w-40 pt-h-32 (160×128px) с pt-overflow-hidden внутри pt-bg-gray-200. Цветной блок имитирует изображение.

pt-object-cover
pt-object-contain
pt-object-fill
pt-object-none
pt-object-scale-down

  <div class="pt-w-full pt-h-full pt-object-cover pt-bg-gradient-to-br pt-from-primary pt-to-purple-600"></div>
</div>
Класс object-fit Поведение
pt-object-covercoverЗаполняет контейнер, обрезая края (сохраняет пропорции)
pt-object-containcontainВмещается целиком, возможно с пустыми полями
pt-object-fillfillРастягивается без сохранения пропорций
pt-object-nonenoneОригинальный размер, без масштабирования
pt-object-scale-downscale-downКак contain, но не увеличивает (только уменьшает)

Ширина и высота

Комбинация pt-w-full pt-h-auto pt-object-cover создаёт адаптивное изображение, которое растягивается на всю ширину родителя, сохраняя пропорции. pt-max-w-full предотвращает выход за границы контейнера.

Адаптивный блок на всю ширину
<div class="pt-max-w-md pt-bg-gray-200 pt-rounded pt-overflow-hidden">
  <div class="pt-w-full pt-h-auto pt-bg-gradient-to-r pt-from-primary pt-to-purple-600 pt-text-white pt-text-center pt-py-16 pt-font-bold pt-text-xl">Адаптивный блок</div>
</div>

Классы для адаптивных изображений:

Класс CSS Назначение
pt-w-fullwidth: 100%Растягивает на всю ширину родителя
pt-h-autoheight: autoАвтоматическая высота (сохраняет пропорции)
pt-max-w-fullmax-width: 100%Не даёт выйти за пределы контейнера
pt-object-coverobject-fit: coverКроп без искажений
pt-py-16padding: 4rem 0Вертикальные отступы для соотношения сторон

Градиентный оверлей

Наложение градиента поверх изображения для читаемости текста. Используется pt-absolute pt-inset-0 для позиционирования, pt-bg-gradient-to-t pt-from-black/70 pt-to-transparent для плавного затемнения снизу вверх, и pt-flex pt-items-end pt-p-4 для размещения текста внизу.

Подпись снизу 
<div class="pt-relative pt-w-64 pt-h-40 pt-bg-gradient-to-br pt-from-primary pt-to-purple-600 pt-rounded-lg pt-overflow-hidden">
  <div class="pt-bg-gradient-to-t pt-from-black/70 pt-to-transparent pt-absolute pt-inset-0 pt-flex pt-items-end pt-p-4">
    <span class="pt-text-white pt-font-bold pt-text-lg">Подпись снизу</span>
  </div>
</div>

Классы для градиентного оверлея:

Класс Назначение
pt-relativeТочка отсчёта для абсолютного позиционирования
pt-absoluteАбсолютное позиционирование оверлея
pt-inset-0Растягивает на весь родитель (top/right/bottom/left: 0)
pt-bg-gradient-to-tГрадиент снизу вверх
pt-from-black/70Начальный цвет с прозрачностью 70%
pt-to-transparentПлавный переход к прозрачности
pt-items-endПрижимает текст к нижнему краю
pt-overflow-hiddenСкрывает выходящие за границы части

Тёмная тема

Изображения и их контейнеры также адаптируются к тёмной теме через pt-dark:* модификаторы. Добавьте pt-dark:bg-gray-800 для фона и pt-dark:border-gray-700 для рамок — контейнеры будут корректно отображаться в обоих режимах.
