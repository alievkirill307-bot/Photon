Навигация [pt]

Компоненты навигации, собранные на pt-* утилитарных классах: меню, табы, хлебные крошки, пагинация, вертикальные и адаптивные навбары — без кастомного CSS.

Горизонтальное меню

Flex-контейнер с pt-flex pt-gap-1 и скруглёнными пунктами. Активный пункт выделен pt-bg-primary pt-text-white, остальные получают hover-эффект через pt-hover:bg-gray-100.

Табы

Горизонтальные табы с нижней границей контейнера pt-border-b pt-border-gray-200. Активная вкладка получает pt-border-b-2 pt-border-primary pt-text-primary, остальные — pt-text-gray-500 pt-hover:text-gray-700. Для отключённого пункта используем pt-text-gray-400 pt-cursor-not-allowed.

Хлебные крошки

Навигационная цепочка из ссылок, разделённых символом /. Контейнер: pt-flex pt-items-center pt-gap-1 pt-text-sm pt-text-gray-500. Разделитель: pt-text-gray-300. Последний элемент — pt-text-gray-700 pt-font-medium.

Главная / Категория / Подкатегория / Текущая 
<div class="pt-flex pt-items-center pt-gap-1 pt-text-sm pt-text-gray-500">
  <a href="#" class="pt-text-primary pt-hover:underline">Главная</a>
  <span class="pt-text-gray-300">/</span>
  <a href="#" class="pt-text-primary pt-hover:underline">Категория</a>
  <span class="pt-text-gray-300">/</span>
  <span class="pt-text-gray-700 pt-font-medium">Текущая</span>
</div>

Альтернативный разделитель — > или любой другой символ, его можно заменить на иконку SVG с pt-w-4 pt-h-4.

Пагинация

Номера страниц с активным состоянием и многоточием. Активный: pt-bg-primary pt-text-white. Остальные: pt-text-gray-600 pt-hover:bg-gray-100 pt-transition. Многоточие: pt-px-2 pt-text-gray-400.

1 2 3 ... 10 
<div class="pt-flex pt-items-center pt-gap-1">
  <span class="pt-px-3 pt-py-1 pt-rounded pt-text-sm pt-bg-primary pt-text-white">1</span>
  <a href="#" class="pt-px-3 pt-py-1 pt-rounded pt-text-sm pt-text-gray-600 pt-hover:bg-gray-100 pt-transition">2</a>
  <a href="#" class="pt-px-3 pt-py-1 pt-rounded pt-text-sm pt-text-gray-600 pt-hover:bg-gray-100 pt-transition">3</a>
  <span class="pt-px-2 pt-text-gray-400">...</span>
  <a href="#" class="pt-px-3 pt-py-1 pt-rounded pt-text-sm pt-text-gray-600 pt-hover:bg-gray-100 pt-transition">10</a>
</div>

Стрелки навигации (назад/вперёд) добавляются теми же отступами и pt-font-bold для символов &laquo; и &raquo;:

&laquo; 1 2 3 ... 10 &raquo;

Адаптивное меню

Скрываем навигационные ссылки на мобильных устройствах через pt-hidden pt-md:flex и показываем иконку бургера только на маленьких экранах через pt-md:hidden.

Logo
Главная О нас Контакты 
<nav class="pt-flex pt-items-center pt-justify-between pt-px-4 pt-py-3 pt-bg-white pt-dark:bg-gray-800 pt-shadow-sm pt-rounded-lg">
  <div class="pt-font-bold">Logo</div>
  <!-- Показываем только на md и выше -->
  <div class="pt-hidden pt-md:flex pt-items-center pt-gap-4">
    <a href="#" class="pt-text-sm pt-text-gray-600">Главная</a>
    <a href="#" class="pt-text-sm pt-text-gray-600">О нас</a>
  </div>
  <!-- Только на мобильных -->
  <button class="pt-md:hidden pt-p-2">
    <!-- иконка бургер -->
  </button>
</nav>

Для полноценного адаптивного меню добавьте pt-flex-col pt-md:flex-row и скрывайте/показывайте блок через pt-hidden pt-md:flex.

Тёмная тема

Модификатор pt-dark:* адаптирует навигационные компоненты под тёмную тему. Основные паттерны:

Элемент Светлая тема Тёмная тема
Фон менюpt-bg-whitept-dark:bg-gray-800
Текст пунктовpt-text-gray-600pt-dark:text-gray-300
Hover фонpt-hover:bg-gray-100pt-dark:hover:bg-gray-700
Границы табовpt-border-gray-200pt-dark:border-gray-700
Теньpt-shadow-smpt-dark:shadow-gray-900 (если поддерживается)
