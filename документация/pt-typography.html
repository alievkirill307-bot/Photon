Типографика [pt]

Утилитарные классы pt-* для типографики — размеры, насыщенность (font-weight), цвета, межстрочный интервал (line-height), межбуквенное расстояние (letter-spacing), выравнивание и оформление текста. Все классы работают независимо и комбинируются друг с другом без ограничений.

Размеры текста

Управляйте размером шрифта через классы pt-text-{size}. Доступны 8 размеров от xs до 4xl:

pt-text-xs (0.75rem) — экстра-маленький текст для подписей и меток pt-text-sm (0.875rem) — маленький текст для вспомогательной информации pt-text-base (1rem) — базовый размер основного текста pt-text-lg (1.125rem) — крупный текст для акцентов pt-text-xl (1.25rem) — очень крупный текст для подзаголовков pt-text-2xl (1.5rem) — заголовочный размер h3/h4 pt-text-3xl (1.875rem) — заголовочный размер h2 pt-text-4xl (2.25rem) — крупный заголовок h1

Насыщенность (font-weight)

Классы pt-font-{weight} меняют жирность шрифта. Доступны все 9 значений от 100 до 900:

pt-font-thin (100) — тонкое начертание, почти невидимое pt-font-extralight (200) — экстра-лёгкое начертание pt-font-light (300) — лёгкое начертание pt-font-normal (400) — нормальное начертание по умолчанию pt-font-medium (500) — среднее начертание pt-font-semibold (600) — полужирное начертание pt-font-bold (700) — жирное начертание pt-font-extrabold (800) — экстра-жирное начертание pt-font-black (900) — максимально жирное, самая толстая насыщенность

Межстрочный интервал (line-height)

Классы pt-leading-* управляют расстоянием между строками текста. Полезно для улучшения читаемости длинных текстов:

pt-leading-none (1)
Строка первая с минимальным межстрочным интервалом.
Вторая строка почти вплотную прилегает к первой.
Подходит для заголовков и коротких надписей.
pt-leading-tight (1.25)
Компактный межстрочный интервал.
Используется для заголовков и плотных блоков текста.
Строки расположены близко, но не слипаются.
pt-leading-snug (1.375)
Умеренно-компактный интервал.
Золотая середина между tight и normal.
Хорошо смотрится в карточках и компактных списках.
pt-leading-normal (1.5)
Стандартный межстрочный интервал.
Используется в большинстве текстов по умолчанию.
Обеспечивает комфортное чтение основного контента.
pt-leading-relaxed (1.625)
Расслабленный межстрочный интервал.
Хорошо читается в длинных статьях и документации.
Даёт глазу больше пространства между строками.
pt-leading-loose (2)
Свободный межстрочный интервал.
Максимальное расстояние между строками.
Используется в поэзии или для визуального выделения. 
<p class="pt-leading-none">line-height: 1</p>
<p class="pt-leading-tight">line-height: 1.25</p>
<p class="pt-leading-snug">line-height: 1.375</p>
<p class="pt-leading-normal">line-height: 1.5</p>
<p class="pt-leading-relaxed">line-height: 1.625</p>
<p class="pt-leading-loose">line-height: 2</p>

Межбуквенное расстояние (letter-spacing)

Классы pt-tracking-* управляют расстоянием между символами (трекингом). Полезно для заголовков и акцентного текста:

pt-tracking-tighter — очень плотный трекинг, буквы сжаты pt-tracking-tight — плотный трекинг, компактное расположение pt-tracking-normal — стандартный трекинг (по умолчанию) pt-tracking-wide — широкий трекинг, буквы разнесены pt-tracking-wider — ещё шире, заметное расстояние pt-tracking-widest — максимальный трекинг, буквы далеко друг от друга

<span class="pt-tracking-tight">Плотно</span>
<span class="pt-tracking-normal">Нормально</span>
<span class="pt-tracking-wide">Широко</span>
<span class="pt-tracking-wider">Ещё шире</span>
<span class="pt-tracking-widest">Максимально</span>

Выравнивание текста

Классы pt-text-{align} для горизонтального выравнивания текста внутри блока:

pt-text-left — Текст прижат к левому краю. Используется по умолчанию для большинства языков с письмом слева направо. Обеспечивает естественное чтение.

pt-text-center — Текст выровнен по центру. Подходит для заголовков, коротких фраз, призывов к действию и подписей под изображениями.

pt-text-right — Текст прижат к правому краю. Удобно для подписей, дат, авторства и интерфейсов с письмом справа налево.

pt-text-justify — Текст выровнен по ширине. Обе стороны абзаца образуют ровные края за счёт автоматического изменения межсловных пробелов. Используется в книгах и газетах.

<p class="pt-text-left">Слева</p>
<p class="pt-text-center">По центру</p>
<p class="pt-text-right">Справа</p>
<p class="pt-text-justify">По ширине</p>

Оформление текста

Классы для подчёркивания, зачёркивания, регистра и наклона текста:

pt-underline — подчёркнутый текст (text-decoration: underline) pt-line-through — зачёркнутый текст (text-decoration: line-through) pt-uppercase — все буквы ЗАГЛАВНЫЕ (text-transform: uppercase) pt-lowercase — все буквы строчные (text-transform: lowercase) pt-capitalize — Каждое Слово С Большой Буквы (text-transform: capitalize) pt-italic — курсивный текст (font-style: italic) pt-not-italic — отмена курсива (font-style: normal) 
<span class="pt-underline">Подчёркнутый</span>
<span class="pt-line-through">Зачёркнутый</span>
<span class="pt-uppercase">ВЕРХНИЙ РЕГИСТР</span>
<span class="pt-lowercase">нижний регистр</span>
<span class="pt-capitalize">Каждое Слово</span>
<span class="pt-italic">Курсив</span>

Truncate (обрезка текста)

Класс pt-truncate обрезает текст, выходящий за границы блока, добавляя многоточие в конце. Сочетает overflow: hidden, text-overflow: ellipsis и white-space: nowrap:

pt-truncate — это очень длинный текст, который не помещается в одну строку и будет обрезан с многоточием в конце строки 
<div class="pt-truncate" style="max-width: 300px;">
  Очень длинный текст, который будет обрезан с многоточием...
</div>

Стили списков

Классы pt-list-* управляют маркерами списков:

pt-list-none — маркеры отсутствуют
  • Элемент списка без маркера
  • Удобно для навигационных меню
  • Чистый внешний вид
pt-list-disc — маркеры в виде точек
  • Первый элемент с точкой
  • Второй элемент с точкой
  • Третий элемент с точкой
pt-list-decimal — нумерованный список
  1. Первый шаг инструкции
  2. Второй шаг инструкции
  3. Третий шаг инструкции
<ul class="pt-list-none"><li>Без маркера</li></ul>
<ul class="pt-list-disc"><li>С точкой</li></ul>
<ol class="pt-list-decimal"><li>Нумерованный</li></ol>

Пример статьи

Комбинация типографических утилит для создания реалистичного превью статьи:

Заголовок статьи с использованием pt-утилит

Автор: Иван Петров · 12 июня 2026 · pt-текст

Первый абзац статьи с демонстрацией комфортного межстрочного интервала pt-leading-relaxed, который делает чтение длинных текстов приятным и ненапряжным. Все утилиты pt-* работают в комбинации друг с другом, позволяя гибко настраивать внешний вид без единой строчки кастомного CSS.

Второй абзац продолжает тему и показывает, как утилиты pt-text-gray-700 для цвета, pt-leading-relaxed для интервала и pt-mb-4 для отступов работают вместе, создавая целостную и читаемую типографическую систему.

«Хорошая типографика — это 95% хорошего дизайна. С pt-классами она становится доступной каждому»
<article class="pt-max-w-2xl pt-mx-auto">
  <h1 class="pt-text-3xl pt-font-bold pt-tracking-tight pt-mb-2">Заголовок статьи</h1>
  <p class="pt-text-gray-500 pt-text-sm pt-mb-1 pt-font-medium">Автор ...</p>
  <p class="pt-text-gray-700 pt-leading-relaxed pt-mb-4">Первый абзац...</p>
  <blockquote class="pt-border-l-4 pt-border-primary pt-pl-4 pt-text-gray-500 pt-italic">Цитата</blockquote>
</article>

Тёмная тема

Все типографические классы поддерживают тёмную тему через префикс pt-dark:. Например, pt-dark:text-white сделает текст белым в тёмной теме. Это позволяет создавать адаптивные интерфейсы без медиа-запросов:

Этот текст — серый в светлой теме и светло-серый в тёмной (pt-dark:text-gray-300)

Акцентный цвет с адаптацией под тёмную тему

Основной текст, тёмный в светлой теме и белый в тёмной

<p class="pt-text-gray-700 pt-dark:text-gray-300">Адаптивный серый</p>
<p class="pt-text-primary pt-dark:text-primary-300">Адаптивный акцент</p>
<p class="pt-text-gray-900 pt-dark:text-gray-100">Адаптивный основной</p>
