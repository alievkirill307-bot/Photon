Секундомер

Полнофункциональный секундомер с точностью до сотых долей секунды. Поддерживает старт, паузу, сброс и запись кругов. Работает полностью на JavaScript без внешних зависимостей.

Демо

Нажмите «Старт» для запуска, «Круг» для записи промежуточного времени, «Пауза» для остановки и «Сброс» для обнуления.

00:00.00

HTML-код

Скопируйте этот код на свою страницу — секундомер заработает сразу после подключения photon-kit.js. Убедитесь, что у корневого элемента указан атрибут data-stopwatch.

<div class="stopwatch" data-stopwatch>
  <div class="stopwatch__display">00:00.00</div>
  <div class="stopwatch__controls">
    <button class="btn btn--primary" data-stopwatch-start>Старт</button>
    <button class="btn btn--outline" data-stopwatch-pause>Пауза</button>
    <button class="btn btn--outline" data-stopwatch-reset>Сброс</button>
    <button class="btn btn--outline" data-stopwatch-lap>Круг</button>
  </div>
  <div class="stopwatch__laps"></div>
</div>

Атрибуты

Секундомер управляется через data-атрибуты на элементах управления. Ниже приведено описание каждого атрибута.

АтрибутНазначение
data-stopwatchКорневой элемент — активирует секундомер
data-stopwatch-startКнопка запуска
data-stopwatch-pauseКнопка паузы
data-stopwatch-resetКнопка сброса
data-stopwatch-lapКнопка записи круга
