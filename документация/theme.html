Тема оформления

Photon поддерживает светлую и тёмную тему через CSS-переменные. По умолчанию используется светлая тема. Для переключения на тёмную добавьте класс theme-dark к элементу <body> (или любому другому контейнеру).

Как использовать

Светлая тема (по умолчанию)

Просто подключите photon-kit.css — светлая тема включена по умолчанию. Никаких дополнительных классов не требуется.

<body>
  <!-- Светлая тема активна по умолчанию -->
</body>

Тёмная тема

Добавьте класс theme-dark к <body>, чтобы включить тёмную тему. Все дочерние элементы автоматически применят тёмные цвета.

Пример тёмной темы (наведите курсор для просмотра):

Тёмная тема Активно Внимание 
<body class="theme-dark">
  <!-- Тёмная тема для всей страницы -->
</body>

Принудительная светлая тема

Если вы используете тёмную тему на всём сайте, но хотите переключить конкретный блок обратно на светлую — добавьте класс theme-light.

<body class="theme-dark">
  <div class="theme-light">
    <!-- Этот блок будет светлым -->
  </div>
</body>

CSS-переменные темы

Photon использует CSS-переменные для всех цветов. Вы можете переопределить любую из них в своём CSS, чтобы создать кастомную тему.

Создание своей темы

Вы можете создать собственную тему, переопределив переменные внутри CSS-класса:

.theme-ocean {
  --pk-bg: #e8f4f8;
  --pk-bg-alt: #d1ecf1;
  --pk-surface: #ffffff;
  --pk-text: #0c5460;
  --pk-primary: #17a2b8;
  --pk-border: rgba(0, 100, 120, 0.12);
}

/* Использование: <body class="theme-ocean"> */
