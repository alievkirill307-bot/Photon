Подсказки, выпадающие списки и меню

Вспомогательные компоненты для взаимодействия: подсказки (tooltips), поповеры (popovers), выпадающие списки (dropdowns) и меню (menus).

Подсказки (Tooltip)

Используйте класс tooltip с атрибутом data-tooltip для отображения подсказки. Позиция задаётся классами tooltip--top (по умолчанию), tooltip--bottom, tooltip--left, tooltip--right.

Наведи на меня Снизу Слева Справа 
<span class="tooltip tooltip--top" data-tooltip="Подсказка сверху">Наведи</span>
<span class="tooltip tooltip--bottom" data-tooltip="Подсказка снизу">Снизу</span>
<span class="tooltip tooltip--left" data-tooltip="Подсказка слева">Слева</span>
<span class="tooltip tooltip--right" data-tooltip="Подсказка справа">Справа</span>

Подсказки работают на любых элементах: кнопках, ссылках, иконках.

<button class="btn btn--primary tooltip" data-tooltip="Сохранить изменения">
  Сохранить
</button>
<a href="#" class="tooltip" data-tooltip="Перейти на главную">Главная</a>

Поповеры (Popover)

Поповер — расширенная подсказка с заголовком и контентом. Используйте класс popover.

Заголовок

Содержимое поповера с дополнительной информацией.

<div class="popover popover--top">
  <button class="btn btn--primary">Открыть поповер</button>
  <div class="popover__content">
    <div class="popover__header">Заголовок</div>
    <div class="popover__body">
      <p>Содержимое поповера.</p>
    </div>
  </div>
</div>

Выпадающие списки (Dropdown)

Используйте dropdown с dropdown__menu для создания выпадающего списка. По умолчанию открывается при наведении.

Профиль Настройки Помощь
Выйти
Действие 1 Действие 2 Действие 3 
<div class="dropdown">
  <button class="btn btn--primary">Меню ▾</button>
  <div class="dropdown__menu">
    <a href="#" class="dropdown__item">Профиль</a>
    <a href="#" class="dropdown__item">Настройки</a>
    <a href="#" class="dropdown__item">Помощь</a>
    <hr class="dropdown__divider" />
    <a href="#" class="dropdown__item" style="color: var(--pk-error);">Выйти</a>
  </div>
</div>

<!-- Выравнивание вправо -->
<div class="dropdown dropdown--right">
  <button class="btn btn--outline">Действия ▾</button>
  <div class="dropdown__menu">
    <a href="#" class="dropdown__item">Действие 1</a>
    <a href="#" class="dropdown__item">Действие 2</a>
  </div>
</div>

Меню (Menu)

Компонент меню для списка действий. Используйте классы menu, menu__item.

Аккаунт
Профиль Настройки Безопасность
Помощь Выйти 
<div class="menu" style="max-width: 200px; border: 1px solid var(--pk-border); border-radius: var(--pk-radius);">
  <div class="menu__header">Аккаунт</div>
  <a href="#" class="menu__item active">Профиль</a>
  <a href="#" class="menu__item">Настройки</a>
  <a href="#" class="menu__item">Безопасность</a>
  <div class="menu__divider"></div>
  <a href="#" class="menu__item">Помощь</a>
  <a href="#" class="menu__item" style="color: var(--pk-error);">Выйти</a>
</div>
